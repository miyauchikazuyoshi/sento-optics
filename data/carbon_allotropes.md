# 炭素同素体の光学データ集

## 概要
同一元素（C）の異なる同素体における光学特性の系統的整理。
原子種を固定することで、混成軌道（非局在化度δ）と構造次元性Dの効果のみを抽出。

## 理論的予測と検証項目

### P1: δ-バンドギャップ逆相関
非局在化度δの増大に伴いバンドギャップが減少する。
- ダイヤモンド(δ低, Eg=5.47) → C60(δ中, Eg≈2) → グラファイト(δ高, Eg≈0) ✓

### P2: D-光学応答分岐
同じδ(高)でも次元性Dで光学応答が分岐。
- D=2(グラファイト): 黒 + 劈開面光沢
- D=3(仮想的3D sp2炭素 = 金属炭素): 金属光沢
→ 圧力下でのグラファイト→金属転移での反射率変化で検証可能

### P3: グラフェン積層の線形吸収
吸収率 = πα × N（N = 層数）
- N=1: 2.3%
- N=2: 4.6%
- ...
- N≈30: ≈70%（グラファイトの不透明性）
→ 既存データで検証可能

### P4: h-BN対照
グラファイトと同構造(D=2)だが、B-N間の電気陰性度差によりπ電子が局在。
δ低 → バンドギャップ大(≈6eV) → 透明(白)
→ 次元性Dだけでは光学応答を決定できず、δが独立変数であることの証拠

### P5: C60の中間性
0D局在だがπ電子が球面上に部分非局在化。
バンドギャップが可視域(≈2eV)に入り、選択的吸収 → 色。
ダイヤモンド(完全局在→透明)と金属(完全非局在→光沢)の中間状態。

## 定量化に向けた課題

1. δの厳密な定義：各同素体でのδを定量計算する手法の選定
   - 候補：逆参加率(IPR)、Wannier関数の広がり、バンド幅/原子軌道幅比
2. D=0(曲面)の数学的記述：フラーレンの球面非局在化の次元性をどう表現するか
3. カーボンナノチューブのカイラリティ依存性：(n,m)指数とδ×Dの対応関係
