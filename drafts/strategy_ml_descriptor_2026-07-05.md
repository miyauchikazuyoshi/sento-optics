## 1. 中心命題（著者）

> δ_elec と δ_nuc の定義が出さえすれば、あとは物量勝負。試算コストはまだ低い。

検証結果: **正しい。しかも一部は想定より安い**（§3 コスト分解）。

## 2. 前例と売り込み口上

- **d-band center (Hammer–Nørskov 1995)**: 一つの電子構造量が触媒スクリーニングの
  標準 ML 特徴量になった最成功例。
- δ/λ² は「d-band center の**局在版**」: 彼らは「d 状態が**どこに**あるか」、
  我々は「**どれだけ広がっているか**」— 直交する情報（positioning_map.md §2 と整合）。
- ML 的な決定的利点: **SCF 1発の占有波動関数から後処理数分で出る**
  （2026-07-05 の実験 F-0/F-2 で実証済み）。GW・励起状態・遷移行列要素が不要。

## 3. ブレイクスルー候補（近い順）

1. **飽和度 s マップ**（実験C の大規模化）: s = 2m_eE_gξ²/ℏ² は「ギャップ律速か
   コア遮蔽律速か」を1数で分離（Si 0.91 vs Ge 0.16 で実証済み）。
   → 高屈折率・低吸収材料の設計チャート + ML アブレーションのベンチマーク。
   **それ自体が一本の論文**。
2. **超イオン伝導体の副格子 z_nuc**: 固体電解質は「Li 副格子だけ融けた固体」。
   z_nuc を元素種別に分解すれば部分融解の直接の記述子
   → 電池材料スクリーニング。既存パイプラインがほぼそのまま使える。
3. **液体・非晶質の記述子という空白市場**: 現在の材料 ML はほぼ結晶専用
   （グラフNNは格子前提）。(z_nuc, z_elec) は非結晶物質の電子構造記述子の
   空白を埋める。Paper 3 の物語と地続き。
4. 遠い夢: f_ZP の第一原理化による融点のノンフィッティング予測、
   金属-非金属転移（金属水素・warm dense matter）、核量子効果材料の事前同定。

## 5. フェーズ0の具体的手順

1. **一般セル対応**（最初のコーディングタスク）: 現行 exp_F0_zelec.py は立方セル専用。
   非立方への一般化 = b を逆格子基底ベクトルに、λ² をテンソル化
   （SWM Eq. 40–42 の quantum metric テンソルに対応）。
2. **δ_nuc 側**: Togo phonondb（NIMS MDR ホスト、現行URLは要確認）から
   thermal_displacements.yaml を一括取得 → ⟨u²⟩(T)/a² と δ_ZP、f_ZP を抽出
   → (δ_nuc, Θ_D, T_m) テーブル。追加計算ゼロ。
3. **δ_elec 側**: Materials Project API（mp-api、要APIキー）で
   E_g > 0.5 eV・nsites ≤ 20 の軽い系を 50–100 件選定 →
   pymatgen で QE 入力自動生成 → SCF → F-0 後処理 → s マップ v0。
4. **検証セット**: 実験Cの C/Si/Ge + NaCl を再現できることを一般化コードの
   回帰テストにする。
5. **MLIP 試運転**（F-1 の補完を兼ねる）: MACE-MP か CHGNet で液体 Na の軌跡を生成し、
   z_nuc を後処理 → AIMD（F-1）の結果と突き合わせ。一致すれば液体側の
   物量路線が開通、F-1 は「MLIP 検証のground truth」という追加の意義を持つ。

## 6. 「物量に耐える定義」の3要件 — Paper 3 へのフィードバック

1. **パラメータフリー**: 窓・基底・カットオフ依存を潰す。IPR でなく z/λ² ベースを
   選んだ現路線はこの点で正解（脅威0 への対処とも整合）。
2. **無人で回る**: 人間の判断が入らない。Wannier 化不要の F-0 方式（G-shift det）が効く。
3. **失敗の自己検出**: PAW ノルム診断（2p = 0.598 の教訓）のような内部チェックを
   定義の一部として組み込む。

## 8. 理論の役割の再定義（同日追記 — README にも要約を掲載）

**著者の観察**: 「理論から新物質を具体に予想するまでが理論のタスクだったかも
しれないが、それは ML/AI のタスクになっていく」。

精密化と帰結:
- **移るのは補間的予測**（既知の物理の範囲での組み合わせ探索 — GNoME,
  MatterGen, ユニバーサル MLIP が既に人間より速く広い）。
  **外挿的予言**（訓練分布の外の新しい物理 — BCS、トポロジカル絶縁体級の飛躍）は残る。
- 機械が供給できない4つが理論に残り、重要性を増す:
  1. **変数の発明** — どの量で世界を見るか（d-band center、トポロジカル不変量、δ）
  2. **拘束の供給** — 外挿に耐える壁（不等式・総和則。ξ² ≤ ℏ²/2m_eE_g など）。
     ML は分布の外で静かに壊れるが、定理は壊れない
  3. **問いの発明** — 「液体とは何か」はデータセットから出てこない
  4. **理解（圧縮）** — 1万件の予測を一行（「光沢＝局在長の発散」）にすること
- **インターフェースの変化**: 理論の顧客が実験家 → ML パイプラインへ。
  成果物の形式は「この物質はこうなる」という予言から、
  「この量を計算せよ・この不等式を守れ」という API 的仕様へ。
  §6 の3要件（パラメータフリー・無人・自己診断）はその仕様書に他ならない。
- **自己言及**: 本プログラムの2日間のワークフロー（人間の直感と問いが上流、
  AI による文献照合・実装・大量検証が下流、事前登録が保証の形式）は、
  この分業の小規模なリハーサルである。
