# Ising RG: δ (IPR) as RG Flow Variable

**Status: 2026-04-03。予備的数値実験。v3でRGフロー反転を確認。**

## 目的

2D Ising模型のブロックスピン繰り込み群（RG）変換において、
δ（確率分布のIPR）がRGフロー変数として機能するかを検証する。

## 実験の流れ

```
v2（磁化IPR）→ フロー反転せず → レビューで指摘 → δ定義を変更
→ v3（ブロックパターンIPR）→ フロー反転を確認 ✓
```

---

## v2: 磁化IPR（δ_norm = m²）

### 定義

p₊ = +1スピンの割合、p₋ = -1スピンの割合。
IPR = p₊² + p₋²。δ_norm = 2·IPR − 1 = m²。

### 結果（L=64, 128, 256）

| T | δ(L=64) | δ(L=128) | δ(L=256) |
|---|---------|----------|----------|
| 1.50 | 0.973 | 0.973 | 0.973 |
| 2.00 | 0.833 | 0.833 | 0.829 |
| Tc | 0.387 | 0.328 | **0.263** |
| 2.50 | 0.014 | 0.004 | 0.001 |
| 4.00 | 0.001 | 0.000 | 0.000 |

### 判定

| テスト | 結果 |
|--------|------|
| δによる相の識別 | **PASS** — T < TcとT > Tcで6桁以上の差 |
| L=64/128/256 一致 (T ≠ Tc) | **PASS** — 差 < 0.005 |
| フロー方向の反転 | **FAIL** — 全Tでδが粗視化で増加（有限サイズ効果） |
| Tcでのスケール不変性 | **FAIL** — CV: 0.19→0.24→0.29（Lとともに悪化） |
| δ(Tc) の L 依存性 | δ(Tc) ∝ L^{−2β/ν} — 秩序パラメータのスケーリングに従う |

### 失敗の原因（レビューによる指摘）

δ_norm = m² は秩序パラメータの二乗であり、RGフロー変数ではない。
Tcで |m| ∝ L^{−β/ν} なので δ ∝ L^{−2β/ν} → 構造的にスケール不変にならない。

---

## v3: ブロックパターンIPR

### 定義

格子を2×2ブロックに分割。各ブロックは16通りのスピンパターンを取る。
各パターンの出現頻度 p_i を数え、IPR = Σ p_i² を計算。

- 完全秩序（T→0）: 1パターンのみ → IPR = 1
- 完全無秩序（T→∞）: 16パターン等確率 → IPR = 1/16 = 0.0625

### 結果（L=64, 128, 256）

| T | IPR(L=64) | IPR(L=128) | IPR(L=256) |
|---|-----------|------------|------------|
| 1.50 | 0.952 | 0.952 | 0.952 |
| 2.00 | 0.761 | 0.758 | 0.760 |
| 2.17 | 0.609 | 0.614 | 0.612 |
| Tc | 0.422 | 0.423 | **0.387** |
| 2.32 | 0.302 | 0.246 | 0.231 |
| 2.50 | 0.180 | 0.175 | 0.172 |
| 4.00 | 0.083 | 0.081 | 0.077 |

### RGフロー方向

| T | L=256 flow | 方向 |
|---|-----------|------|
| 1.50 | +0.048 | → 秩序 |
| 2.00 | +0.188 | → 秩序 |
| 2.22 | +0.147 | → 秩序 |
| **Tc** | **+0.046** | **≈ 停滞** |
| 2.32 | **−0.051** | **→ 無秩序** |
| 2.37 | **−0.067** | **→ 無秩序** |
| 2.50 | −0.076 | → 無秩序 |
| 4.00 | −0.014 | → 無秩序 |

**フロー方向がTcを境に反転。** v2では不可能だった真のRGフロー反転がv3で実現。

### スケール不変性

| L | CV at Tc (v2 磁化) | CV at Tc (v3 パターン) | 方向 |
|---|-------------------|---------------------|------|
| 64 | 0.190 | **0.134** | v3が良い |
| 128 | 0.240 | 0.171 | v3が良い |
| 256 | 0.290 | **0.163** | v3が良い、かつL↑で改善 |

- v2: CVがLとともに**悪化**（0.19→0.29）→ 固定点から遠ざかる
- v3: CVがL=128→256で**改善**（0.17→0.16）→ 固定点に近づく方向

完全なスケール不変ではないが、v3は正しい方向に向かっている。

### 判定

| テスト | v2 結果 | v3 結果 |
|--------|---------|---------|
| 相の識別 | PASS | PASS |
| L収束 (T ≠ Tc) | PASS | PASS |
| フロー方向の反転 | FAIL | **PASS** |
| Tcでのスケール不変性 | FAIL (CV悪化) | **改善方向** (CV低下) |

---

## 結論

1. **磁化IPR（v2）はRGフロー変数ではない。** 秩序パラメータのスケーリングに従い、
   Tcでスケール不変にならない。ただし相の識別子としては機能する。

2. **ブロックパターンIPR（v3）はRGフロー変数の候補として有望。**
   Tcを境にフロー方向が反転し、スケール不変性がL↑で改善する。

3. **δの定義が結果を決定的に変える。** 同じ物理系でも「何のIPRを取るか」で
   RGフロー変数としての適格性が変わる。局所構造（パターン）の多様性を測る
   IPRは、大域的秩序パラメータ（磁化）のIPRより、RGの粗視化構造と整合する。

4. **今後**: L=512以上でCVがさらに下がるか確認。下がり続ければ
   「パターンIPRは熱力学極限でTcのRG固定点に収束する」と結論できる。
