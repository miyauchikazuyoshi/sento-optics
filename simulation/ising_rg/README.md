# Ising RG: δ (IPR) as RG Flow Variable

**Status: 2026-04-03。予備的数値実験。v3でRGフロー反転を確認。**

## 目的

2D Ising模型のブロックスピン繰り込み群（RG）変換において、
δ（確率分布のIPR）がRGフロー変数として機能するかを検証する。

## 実験の流れ

```
v1（Claude仮想環境）→ 初期傾向を確認、実験設計を決定
→ v2（ローカル環境）→ L=256まで拡張、レビューで構造的問題を特定
→ v3（ローカル環境）→ δ定義を変更、RGフロー反転を確認 ✓
```

---

## v1: 初期探索（Claude仮想環境）

### 実験設計

- **環境**: Claude仮想環境（計算資源に制約あり）
- **系**: 2D Ising模型、L=64、Wolffクラスターアルゴリズム
- **δ定義**: 磁化IPR（p₊² + p₋²）
- **RG**: 2×2多数決ブロックスピン変換
- **温度**: T = 1.5, 2.0, Tc(2.269), 2.5, 3.0, 4.0
- **サンプル**: 50配位、熱平衡化300 Wolffステップ（固定）

### 結果

```
T=1.500: δ flow = -0.006  → ordered
T=2.000: δ flow = -0.037  → ordered
T=  Tc:  δ flow = -0.073  → ordered（停滞気味）
T=2.500: δ flow = +0.529  → disordered
T=3.000: δ flow = +0.337  → disordered
T=4.000: δ flow = +0.081  → disordered
```

### 判定

- **フロー方向**: T < Tcで→ordered、T > Tcで→disordered。初期傾向としてPASS。
- **スケール不変性**: Tcでのδがスケール不変かは確認できず（CV = 0.12、ただしT=1.5のCV=0.02より大きい）。
- **L=128の試行**: 熱平衡化不足（300ステップ固定）でδが3-4倍ずれ、失敗。

### v1の意義

- δがRGフロー変数になり得るという**初期傾向**を示した
- L=128の失敗から**適応的熱平衡化（n_therm ∝ L）の必要性**を特定
- v2の実験設計（温度グリッド、サンプル数、熱平衡化戦略）を決定

---

## v2: 磁化IPR（ローカル環境、L=64/128/256）

### 実験設計

v1の教訓を反映:

- **適応的熱平衡化**: n_therm = factor × L（L=128で5120回、L=256で10240回）
- **系サイズ**: L=64, 128, 256の3サイズで有限サイズ効果を系統的に評価
- **温度グリッド**: 粗い点5つ + Tc近傍の細かい点5つ = 10温度
- **サンプル**: 80-100配位、デコリレーション3-5 Wolffステップ
- **δ定義**: v1と同じ磁化IPR。δ_norm = 2(p₊² + p₋²) − 1 = m²

### 結果

| T | δ(L=64) | δ(L=128) | δ(L=256) |
|---|---------|----------|----------|
| 1.50 | 0.973 | 0.973 | 0.973 |
| 2.00 | 0.833 | 0.833 | 0.829 |
| Tc | 0.387 | 0.328 | **0.263** |
| 2.50 | 0.014 | 0.004 | 0.001 |
| 4.00 | 0.001 | 0.000 | 0.000 |

### 判定

| テスト | 結果 |
|--------|------|
| δによる相の識別 | **PASS** — T < TcとT > Tcで6桁以上の差 |
| L=64/128/256 一致 (T ≠ Tc) | **PASS** — 差 < 0.005（v1のL=128失敗を解消） |
| フロー方向の反転 | **FAIL** — 全Tでδが粗視化で増加（有限サイズ効果） |
| Tcでのスケール不変性 | **FAIL** — CV: 0.19→0.24→0.29（Lとともに悪化） |
| δ(Tc) の L 依存性 | δ(Tc) ∝ L^{−2β/ν} — 秩序パラメータのスケーリングに従う |

### レビューによる指摘

> 磁化は秩序パラメータであって、RGフロー変数とは異なる量。

1. **フロー方向について**: 全Tでδが増加するのは、ブロックスピン変換で
   系が縮小し有限サイズの磁化揺らぎが大きくなるため。
   フロー方向の反転ではなく、**フロー速度の差**が秩序/無秩序を区別している。

2. **δ定義の構造的問題**: δ_norm = m²。Tcでは |m| ∝ L^{−β/ν} なので
   δ ∝ L^{−2β/ν}。**構造的にスケール不変にならない。** L=256で確認済み。

3. **提案**: ローカルブロックパターンの分布のIPR、または
   エネルギー分布のIPRがスケール不変性を示す可能性がある。

### 課題（v3への引き継ぎ）

- δ定義を磁化IPRからブロックパターンIPRに変更する
- 同じ温度グリッド・サンプル数・系サイズで再実験
- フロー方向が**真に反転するか**を検証
- TcでのCVがL↑で**改善するか**を検証

## v3: ブロックパターンIPR（ローカル環境、L=64/128/256）

### 実験設計

v2と同一条件でδ定義のみ変更:

- **δ定義**: 2×2ブロックのスピンパターン（16通り）の出現頻度分布のIPR
  - IPR = Σ p_i²（i = 1..16）
  - 完全秩序: IPR = 1（1パターンのみ）
  - 完全無秩序: IPR = 1/16 = 0.0625（16パターン等確率）
- **その他の条件**: v2と同一（L=64/128/256、10温度点、100サンプル、適応的熱平衡化）

### なぜパターンIPRか

1. **局所構造の多様性**を測る（磁化は大域的秩序パラメータ）
2. RGの粗視化（ブロック変換）と**同じスケール**で定義される
3. |m| ∝ L^{−β/ν} のスケーリングに**支配されない**
4. Rényi-2 エントロピーに対応（δ枠組みの情報理論的基盤と整合）

### 結果

#### RGフロー方向（L=256）

| T | flow | 方向 |
|---|------|------|
| 1.50 | +0.048 | → 秩序 |
| 2.00 | +0.188 | → 秩序 |
| 2.22 | +0.147 | → 秩序 |
| **Tc** | **+0.046** | **≈ 停滞** |
| 2.32 | **−0.051** | **→ 無秩序** |
| 2.37 | **−0.067** | **→ 無秩序** |
| 2.50 | −0.076 | → 無秩序 |
| 4.00 | −0.014 | → 無秩序 |

**Tcを境にフロー方向が反転。** v2では不可能だった真のRGフロー反転がv3で実現。

#### IPR(Tc) の L 依存性

| L | IPR(Tc) | v2のδ(Tc)比較 |
|---|---------|-------------|
| 64 | 0.422 | 0.387 |
| 128 | 0.423 | 0.328 |
| 256 | 0.387 | 0.263 |

v2はδ(Tc)がL↑で単調減少（0.39→0.33→0.26）。
v3はL=64→128で安定（0.422→0.423）、L=256で少し下がる（0.387）。

#### スケール不変性

| L | CV at Tc (v2) | CV at Tc (v3) |
|---|--------------|--------------|
| 64 | 0.190 | **0.134** |
| 128 | 0.240 | 0.171 |
| 256 | 0.290 | **0.163** |

- v2: CVがLとともに**悪化**（0.19→0.24→0.29）→ 固定点から遠ざかる
- v3: L=128→256でCVが**改善**（0.17→0.16）→ 固定点に近づく方向

### 判定

| テスト | v2 (磁化) | v3 (パターン) |
|--------|-----------|-------------|
| 相の識別 | PASS | PASS |
| L収束 (T ≠ Tc) | PASS | PASS |
| フロー方向の反転 | FAIL | **PASS** |
| Tcでスケール不変性 | FAIL (CV悪化) | **改善方向** (CV低下) |

---

## 結論

1. **v1**（仮想環境）で初期傾向を確認し、実験設計を固めた。

2. **v2**（磁化IPR）は相の識別子として機能するが、
   RGフロー変数としては構造的に不適格。
   レビューの指摘（秩序パラメータ ≠ RGフロー変数）が正しかった。

3. **v3**（ブロックパターンIPR）はRGフロー変数の候補として有望。
   Tcでフロー方向が反転し、スケール不変性がL↑で改善する。

4. **δの定義が結果を決定的に変える。** 局所構造の多様性を測るIPRは、
   大域的秩序パラメータのIPRより、RGの粗視化構造と整合する。

5. **今後**: L=512以上でCVがさらに下がるか確認。
   下がり続ければ「パターンIPRは熱力学極限でTcのRG固定点に収束する」と結論できる。
