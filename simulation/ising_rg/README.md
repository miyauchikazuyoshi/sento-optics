# Ising RG: δ as RG Flow Variable（予備的数値実験）

**Status: 2026-04-03。予備的な数値実験。結果は promising だが不十分。**
**2026-04-03 レビューにより一部判定を修正。**

## 目的

2D Ising模型のブロックスピン繰り込み群（RG）変換において、
δ（確率分布のIPR）がRGフロー変数として機能するかを検証する。

## 予測（δ枠組み）

1. T < Tc: δ は粗視化で秩序固定点（δ→1）に流れる
2. T > Tc: δ は粗視化で無秩序固定点（δ→0）に流れる
3. T = Tc: δ はスケール不変（RG固定点）

## 結果（v2, L=64,128）

| テスト | 結果 |
|--------|------|
| δによる相の識別 | **PASS** — δ(T < Tc) >> δ(T > Tc) で6桁以上の差 |
| L=64 vs L=128 一致 | **PASS** — T ≤ Tcで差 < 0.03（熱平衡化成功） |
| フロー方向の反転 | **修正** — 全Tでδが粗視化で増加。方向反転ではなく速度差（下記参照） |
| Tcでのスケール不変性 | **構造的に困難** — 現δ定義では原理的にスケール不変にならない可能性（下記参照） |

## レビューによる修正（2026-04-03）

### フロー方向について

当初「T < Tcで→秩序、T > Tcで→無秩序」と判定したが、
実際には**全温度でδが粗視化とともに増加**している。
これはブロックスピン変換でL→L/2と系が縮小する際、
有限サイズの磁化揺らぎが相対的に大きくなるため。

正確な記述:
- T < Tc: δ の初期値が高く（~0.97）、数ステップで δ=1 に飽和
- T > Tc: δ の初期値が極めて低く（~0.001以下）、粗視化でもゆっくり上昇
- Tc: 中間的な初期値（~0.36）から緩やかに上昇

**δの初期値と増加速度がTcを境に劇的に変わる。**

### δ定義の構造的問題

現在の δ_norm = 2(p₊² + p₋²) − 1 = m² は磁化の二乗。
Tcでは |m| ∝ L^{−β/ν} なので δ ∝ L^{−2β/ν}。
**構造的にスケール不変にならない。** これはL=256にしても解決しない。

磁化は秩序パラメータであって、RGフロー変数とは異なる量。

## 拡張方針: ブロックパターンIPR（v3）

レビューの示唆に基づき、δの定義を**ローカルブロックパターンの分布のIPR**に変更する。

### 原理

2×2ブロックのスピンパターンは16通り（↑↑↑↑, ↑↑↑↓, ...）。
各パターンの出現頻度 p_i を数え、IPR = Σp_i² を計算する。

- 完全秩序（T→0）: 1パターンのみ出現 → IPR = 1
- 完全無秩序（T→∞）: 16パターン等確率 → IPR = 1/16 = 0.0625
- Tc: **パターン分布が自己相似 → IPRがスケール不変になる可能性**

### なぜこちらが良いか

1. 磁化（大域的秩序パラメータ）ではなく**局所的な構造多様性**を測る
2. RGの粗視化（ブロックスピン変換）と同じスケールで定義される
3. 配置空間の Rényi-2 エントロピーに対応（δ枠組みの情報理論的基盤と整合）
4. |m| ∝ L^{−β/ν} のスケーリングに支配されない

### 検証項目

1. ブロックパターンIPRがTcでスケール不変になるか
2. T < Tc で IPR → 1（秩序固定点）に流れるか
3. T > Tc で IPR → 1/16（無秩序固定点）に流れるか
4. フロー方向がTcで**真に反転**するか
