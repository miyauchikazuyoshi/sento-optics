# シミュレーションアルゴリズム仕様

## 目的
炭素同素体の光学特性を、非局在化度δと次元性Dから第一原理的に計算し、
実験値との照合により理論の定量的予測力を検証する。

## Phase 1: δの定量計算

### 1.1 逆参加率（IPR）による非局在化度

各同素体について、価電子の波動関数 ψ_i(r) から逆参加率を計算：

```
IPR_i = ∫ |ψ_i(r)|⁴ dr / (∫ |ψ_i(r)|² dr)²
```

- IPR → 1/V（系の体積）: 完全非局在化
- IPR → 1: 完全局在

δ = 1 - IPR × V（規格化）として非局在化度を定義。

### 1.2 計算手法

- DFT（密度汎関数理論）でバンド構造を計算
- コード候補: VASP, Quantum ESPRESSO, GPAW
- 各同素体の構造最適化後、価電子バンドのIPRを計算
- 交換相関汎関数: PBE（標準）、HSE06（バンドギャップ補正用）

}

## Phase 2: 光学特性の計算

### 2.1 誘電関数 ε(ω) の計算

Random Phase Approximation (RPA) またはBSE（Bethe-Salpeter方程式）で
光学伝導度 σ(ω) を計算し、誘電関数を導出：

```
ε(ω) = 1 + 4πi σ(ω) / ω
```

### 2.2 反射率の計算

Fresnel方程式から反射率R(ω)を導出：

```
R(ω) = |(√ε(ω) - 1) / (√ε(ω) + 1)|²
```

### 2.3 吸収係数の計算

```
α(ω) = 2ω κ(ω) / c
```

ここで κ は消衰係数（ε の虚部から導出）。

## Phase 3: δ-光学特性の写像検証

### 3.1 予測

Phase 1で計算したδ × D と、Phase 2で計算した光学特性の対応を検証。

検証項目：
1. δ と バンドギャップ Eg の逆相関
2. δ × D と 可視域平均反射率 R_vis の関係
3. δ × D と 可視域平均吸収率 α_vis の関係
4. グラフェン積層数 N と 吸収率の線形関係

### 3.2 スケーリング則の探索

δ × D を単一パラメータ Γ = f(δ, D) に統合し、
光学特性が Γ の関数としてユニバーサルスケーリングに乗るかを検証。

候補：
```
Γ = δ × D / D_max  （正規化）
Γ = δ^D            （冪乗則）
Γ = δ × (1 - e^(-D)) （飽和型）
```

## Phase 4: 実験値との照合

### 4.1 照合データ

| 物質 | 計算値 | 実験値 | ソース |
|------|--------|--------|--------|
| ダイヤモンド Eg | HSE06計算 | 5.47 eV | Phillip (1964) |
| C60 Eg | HSE06計算 | 1.6-2.3 eV | 複数文献 |
| グラフェン吸収率 | RPA計算 | πα = 2.3% | Nair et al. (2008) |
| グラファイト面内反射率 | RPA計算 | 波長依存 | Kuzmenko et al. (2008) |
| h-BN Eg | HSE06計算 | ~6.0 eV | 文献値 |

### 4.2 成功基準

- δ-Eg 逆相関の相関係数 |r| > 0.9
- グラフェン積層吸収の線形性 R² > 0.95
- 全同素体の光学応答分類（透明/色/黒/光沢）が δ×D で100%正分類

## 実装ロードマップ

- スケーリング則の探索
- 実験値との照合

- 理論部分の執筆

## リスクと代替案

| リスク | 影響 | 代替案 |
|--------|------|--------|
| DFT計算のバンドギャップ過小評価 | δ-Eg相関が歪む | HSE06/GW補正 |
| IPRがδの良い指標でない | 理論の定量化失敗 | Wannier関数広がり、バンド幅比 |
| 計算リソース不足 | 全同素体の計算不能 | ダイヤモンド+グラフェン+h-BNの3点に絞る |
| C60/CNTの大規模計算 | 収束困難 | 文献値のδ推定で代替 |
