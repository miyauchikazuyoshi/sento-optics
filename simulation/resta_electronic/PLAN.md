## F-0: 既存 wfc から z_elec を1点計算（後処理のみ、今すぐ）

**目的**: 液体 Na（中断時スナップショット）の電子側 Resta 位相を初計算。
「液体金属の価電子は非局在（z_elec ≈ 0）、semicore は局在（|z| ~ 1）」の
コントラストを1枚で示す。

**入力**: aimd/tmp/Na_liquid.save/wfc1.hdf5（Γ点、gamma_only、355バンド、
24,659 平面波、Miller 指数つき — 2026-07-05 に構造確認済み）

**方法**: 平面波基底では e^{i b·r}（b = 最小逆格子ベクトル）は G シフト。
M_ij = Σ_G c_i*(G) c_j(G−b)、z = det(M_occ)。占有 288 バンド
（Na spn: 2s²2p⁶3s¹ = 9e/原子 × 64 = 576e）。

**事前判定基準**:
- F0-1: 全占有 det の |z_elec| < 0.1（金属的）
- F0-2: semicore ブロック（下位256バンド）の |z|^(1/N) が 3s ブロックより明確に大きい
  （局在/非局在コントラスト、バンドあたり幾何平均で比較）
- F0-3: 占有数を ±2 バンド振っても F0-1 の結論が変わらない（頑健性）

**明記する近似**: (i) PAW augmentation 補正なし（擬波動関数のみ）、
(ii) 金属の部分占有を整数占有で近似、(iii) 単一スナップショット（時間平均なし）。
定性デモとしては十分、論文数値には F-1 の時間平均＋PAW 検討が必要。

## F-2: 絶縁対照 — NaCl 結晶で z_elec 有限（半日）

液体水 AIMD は 16GB では非現実的なので、**絶縁対照は NaCl 結晶の SCF 1発**で代替。
- 2×2×2 conventional スーパーセル（Na32 + Cl32 = 64 原子、a=5.64 Å → L=11.28 Å）、
  Γ点、occupations='fixed'（絶縁体）、SCF のみ → F-0 と同じ後処理

**事前判定基準（実行前に精密化、2026-07-05）**
絶縁体でも多体 |z| はサイズと共に指数的に小さくなるため、「桁で大きい」ではなく
物質固有量で判定する:
- F2-1: λ²(NaCl) < λ²(液体Na)/2 （< 2.2 Å²）— 絶縁体の局在長は有限で小さい
- F2-2: Cl 3p 価電子帯（上位96バンド）の幾何平均 |z|^(1/N) > 0.8
  （液体 Na 3s の 0.40 の2倍近い局在）
- F2-3: Na semicore は液体 Na と同値（PAW 補正後 ~0.99）— コア局在の系非依存性

これで「2枚看板」の金属側/絶縁側コントラストが固体レベルで完成。

## F-3b（オプション）: twist スキャンで発散傾向

### F-2 結果（判定: F2-1 / F2-2 / F2-3 全 PASS）

SCF: 64原子 NaCl、10 iteration で収束、ギャップ 5.1 eV（PBE、絶縁体確認）。実行 ~65分。

- **F2-1 PASS**: λ²(NaCl) = **1.033 Å²、3方向完全等方**（立方対称の再現 = 良い内部診断）
  ← 液体 Na の 4.39 Å²（N=64 での発散量の値）の 1/4。
- **F2-2 PASS**: Cl 3p 価電子帯の補正後幾何平均 **0.934**（局在価電子）
  ← 液体 Na 3s の 0.40 との明確なコントラスト。
- **F2-3 PASS**: Na semicore = 0.984 — 液体 Na 中（0.992）とほぼ同値。
  **コアの局在は環境（金属液体/イオン結晶）に依らない** — δ の階層性の系横断確認。
- ノルム診断: Na 2p = 0.621（液体 Na の 0.598 と同じ PAW 欠損 → ブロック同定も整合）。

**これで「2枚看板」の固体レベル対照が完成**:
金属（液体Na）= |z| ~ 10⁻⁵⁷・λ² は N と共に発散 ／ 絶縁体（NaCl）= λ² 有限・等方・~1 Å²。

### F-0 結果（判定: F0-1 PASS / F0-2 FAIL→診断済み / F0-3 PASS）

- **F0-1 PASS**: 全占有 |z_elec| ~ 10⁻⁵⁷（3方向とも）。**液体 Na の電子側は
  完璧に金属的（非局在）** — 4象限右上（電子×有限温度）の初の実データ点。
  λ² 換算 ~4.4 Å²（N=64 有限系での見かけ値。熱力学極限では発散する量）。
- **F0-3 PASS**: 占有バンド数 ±2 で結論不変。
- **F0-2 は事前基準（幾何平均比 >3倍）では FAIL — 原因2つを診断**:
  1. **PAW ノルム欠損**: 2p バンドの擬WFノルムは 0.598（2s: 0.998, 3s: 1.002）。
     未補正のブロック det は augmentation 分だけ過小評価される
     （0.598^(192/256)=0.68 が実測 0.6745 と一致 — 定量的に説明済み）。
  2. 対角ノルム補正後は **semicore 0.9922 vs 3s 0.40（比 2.4-2.6x）** —
     「コア局在・価電子非局在」の物理は明確に回復。「>3倍」は幾何平均の
     スケール感を読み違えた基準設定で、コントラスト自体は成立。
- 教訓: F-1 の論文級数値には PAW augmentation の真面目な取り扱い
  （少なくとも対角補正、理想は S 行列込み）が必要。
