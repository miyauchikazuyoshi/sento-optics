## 実験A: 結果と解釈

**事前判定基準**: |z1|_solid(Bragg) / |z1|_liquid(k~2Å⁻¹) > 10

**結果: 8.8 → 形式上 FAIL。ただし原因を特定済み（方法論の失敗ではない）。**

1. **モック bcc 固体側は完璧**: Bragg 波数 (n=6, k=2.93 Å⁻¹) で
   |z1| = 0.8095、Debye-Waller 理論値 exp(−G²σ²/2) = 0.807 と3桁一致。
   他の n ではゼロ。**核版 Resta 位相が DW 因子として機能することの数値実証。**
2. **液体側の残留 (n=4–6 で ~0.09–0.13) は単一凍結配置のショットノイズ**:
   - 軌跡は 0.166 ps しかなく、液体Naの構造緩和 (~1 ps) より遥かに短い。
     拡散距離 √(6Dt) ≈ 0.6 Å — 原子はほぼ初期配置のまま（A-2 で時間分解
     しても横ばい）。
   - 初期配置は最小距離制約付きランダム充填 (generate_liquid_config.py)。
     初期スナップショット単体の |z1| ≈ 0.09–0.12 は 1/√64 = 0.125 と整合。
   - つまり時間平均が効いておらず、実効独立サンプル数 ≈ 1。
3. **副産物**: 平衡液体では並進対称性により z1 は厳密にゼロなので、
   |z1| の時間平均の収束は「配置空間のエルゴード的探索」のモニタになる。
   RESUME_STATE.md の「平衡化途中」を z_nuc が独立に定量検出した形。

**結論**: 方法論は機能。液体側の決着は実験B（十分に平衡化できる古典MD）へ。

## 実験C: 結果

**全ての絶縁体・半導体で SWM Eq. 52 が成立**（Paper 1 の Ω 計算が
定理と整合する独立サニティチェック）。

| System | ξ²_xx (Å²) | E_g^dir (eV) | 飽和度 s (dir) |
|--------|-----------|--------------|----------------|
| Si | 1.016 | 3.40 | **0.907** |
| Diamond | 0.318 | 7.30 | 0.609 |
| Ge | 0.758 | 0.80 | **0.159** |

**発見**: 飽和度に大きな系統差。
- Si はほぼ飽和 (0.91) — 価電子はギャップが許す限界近くまで非局在化
- Ge は 0.16 — ギャップは小さいのに Ω が収縮 = **d 遮蔽律速**
  （wannier_carbon README の発見5「3d¹⁰ が sp³ を25%収縮」と定量的に整合）

→ **飽和度 s = ξ²·2m_eE_g/ℏ² は「ギャップ律速か、コア遮蔽律速か」を
分離する新しい記述子候補**。Paper 1 の δ–E_g 相関 (r=−0.86) は
「不等式の飽和度の系統性」として定理の言葉で書き直せる。

| 判定 | 基準 | 結果 | 判定 |

| B2 | 平衡液体で全 n の \|z1\| < 2×ノイズフロア | max 0.0080 < 0.0088 | **PASS** |

## 実験Bのデバッグ記録（失敗も記録する）

1. **初回 run**: 力の符号バグ（F = +fmag·d、正しくは −fmag·d）で全状態点が
   「反転LJ流体」化。固体対照の MSD 9.6σ² で発覚。2粒子テストで修正確認。
2. **2回目 run**: 固体 \|z1\| = 0.26（期待 ~0.7）。原因は **Langevin 熱浴が
   運動量を保存しない**こと — 結晶全体の重心ランダムウォークが位相 e^{iGx}
   を回し、時間平均を偽って減衰させる。
3. **3回目 run（最終）**: unwrap 軌跡から重心ドリフトを除去して全 PASS。

**教訓（z_nuc 測定の設計注意、AIMD にも適用）**:
- 運動量非保存サーモスタット（Langevin 等）では**重心補正が必須**
- \|z1\| 時間平均の収束は配置空間のエルゴード的探索のモニタになる
  （短すぎる軌跡は実験Aのように「凍結配置のショットノイズ」を残す）

## 実験D: 同位体プローブ — 結果（2026-07-05、判定 D1/D2/D3 全 PASS）

モデル: 融解条件 δ_c = A·T_m + B/√M_eff（ゼロ点項のみ質量依存）。
融点の同位体シフトから**融解時の核非局在化のゼロ点寄与率 f_ZP** を較正:

| 系 | ΔT_m | f_ZP |
|----|------|------|
| H₂/D₂ | +4.77 K (+34%) | **53.8%**（融解の過半がゼロ点駆動 = de Boer 量子固体） |
| Ne20/Ne22 | +0.15 K | 11.6% |
| H₂O/D₂O (libration 解釈) | +3.82 K | 4.6% |
| H₂O/D₂O (並進解釈) | 同上 | 21.4% |

- 序列 H₂ > Ne > H₂O が de Boer Λ* の量子性序列と単調（D2 PASS）
- 水の2解釈（H 質量 vs 分子質量）の判別は HDO・同位体熱容量で可能 → Paper 4 の設計対象
- **表面張力の符号パズル**: 絶対比較では γ_D₂O < γ_H₂O（素朴予測と逆）だが、
  対応状態補正（Guggenheim, T_c の同位体差を除去）で γ0_D₂O/γ0_H₂O = **+0.39%**
  — 「D₂O の方が凝集的」が回復（D3 PASS）。**同位体プローブは対応状態で比較せよ**。
- 注意: NQE 同位体効果自体は確立分野。ここの新規性は「δ 言語での較正プロトコル」。
  文献値は論文使用前に一次出典で要再確認。

## 実験E: 方向分解 λ — 結果（2026-07-05、E1/E2 PASS・E3 FAIL→診断済み）

RS (1999) が「expect」と書いただけで計算しなかった λ の異方性を TB で数値化。

- **E1 PASS**: 1D 二量化鎖 λ²=0.531 に収束・一様鎖（金属）80→1080 と発散
  — RS Fig.2 の定性再現、実装検証
- **E2 PASS（本丸）**: 理想化グラファイト（面内一様金属 × 層間 t_z=0）で
  **λ_z² = 0（数値的厳密ゼロ, −9.6e-16）かつ λ_x² が 20→39 と発散**
  → **「D_eff = λ が発散する方向の数」の初の明示的数値実証**（D_eff = 2）
- **E3 FAIL（正直な記録）**: 弱い t_z (0.05–0.2) での λ_z² 増大の単調性は不成立。
  診断: kz バンド幅 (~2t_z) が有限系の k 間隔 (2π/12≈0.52) 未満で
  **kz 分散が解像できない**（フェルミ面シェル効果）。
- **E-2 追試**: t_z=0.5（kz 幅 ~1.0 > 0.52）でも単一 twist では非単調
  （2.4→3.4→2.8→7.9、傾向は明確に増大側）。滑らかな結果には
  **twist 平均が必要 — SWM が Eq. 32 で境界条件平均を定義に入れた理由を体感**。
  twist 平均版は将来課題。
