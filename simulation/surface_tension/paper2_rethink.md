**Status: 2026-04-09。レビュー対応から生じた再考。**

---

## 背景

レビューでn_ws^{1/3}テーブルの誤りが発覚し、正しい値で相関を再計算した結果:

| 相関 | 正しい値 | 旧値（誤り） |
|------|---------|------------|
| δ_all vs n_ws^{1/3} | **r = 0.14** | r = 0.20 |
| δ_val vs n_ws^{1/3} | **r = 0.26** | r = 0.33 |
| δ_val vs midpoint density | **r = 0.89** | r = 0.89（変わらず） |
| sp/d分離（14金属スラブ） | **p = 0.00008** | （変わらず） |

**δは中間点密度を強く予測する（r=0.89）が、Miedemaのn_wsは予測しない（r=0.26）。**

## 問い

このギャップの原因は何か？ どうすれば埋められるか？

---

## 因子の候補（優先順）

### 1. d電子スクリーニング（最有力）

d電子は局在（δ低い）だが、核電荷をスクリーニングしてsp電子を
セル境界に押し出す → n_wsが上がる。δは直接的な非局在化を測り、
n_wsはスクリーニングの間接効果を含む。

**検証A: 軌道分解δ（δ_s, δ_p, δ_d）**
- PySCFで計算可能（既存test7_extended.pyを拡張）
- 各軌道成分のIPRを分離して相関を取る
- δ_spだけでn_wsと相関するか？

**検証B: δ_sp + d充填度の2変数回帰**
- 既存データだけで即実行可能
- n_ws ≈ α·δ_sp + β·N_d + γ
- r が 0.26 → 0.7+ に改善すれば、スクリーニングが原因と確定

### 2. Miedema n_ws自体の定義問題

n_wsは「合金生成熱をフィットするパラメータ」であり、
実際に電子密度を測定したわけではない。
n_wsが物理的な境界密度と一致するかは自明ではない。

**検証C: WS境界密度の第一原理計算**
- QEスラブ計算の既存データからWS境界上の密度を抽出
- または bulk DFT で Voronoi セル境界の密度を直接計算
- 計算した境界密度 vs Miedema n_ws の相関を取る
- 一致すればn_wsは実際の境界密度。不一致ならn_ws自体が別の物理。

### 3. 結晶構造効果

n_wsはWS セルの形状に依存。BCC/FCC/HCPで境界面積・距離が違う。
δはセル形状を知らない。

**検証D: 同一元素の異なる構造**
- Fe: BCC(110) vs FCC(111) で n_mid/n_bulk を比較
- 構造を変えてもδは同じ → 境界密度だけが変われば構造効果

### 4. 相対論効果

重元素（Ag, Au）で相対論的s収縮・d膨張がδの値を変える。

**検証E: Ag の相対論 vs 非相対論**
- PySCFで scalar relativistic vs non-relativistic のδ比較
- 差がn_ws乖離に対応するか

### 5. コア-価電子境界の曖昧さ

エネルギー窓の選び方でセミコア状態の扱いが変わる。

**検証F: エネルギー窓の系統的変化**
- Paper 2の既存ILDOS感度テストを14金属に拡張
- 窓を変えるとsp/d比率やn_ws相関がどう変わるか

## 主張の再構築案

**主張1（強い、数値的に支持）:**
> δ_IPRは原子間境界電子密度を予測する（r = 0.89）。
> sp金属とd金属で明確に分離される（p = 0.00008）。

**主張2（中程度、定性的に支持）:**
> Miedemaのn_wsは、境界電子密度と概念的に対応する。
> n_wsが「価電子非局在化に関連する」という解釈は整合的。

**主張3（正直、定量化が必要）:**
> δとn_wsの直接相関は弱い（r = 0.26）。
> ギャップの主因はd電子スクリーニング（仮説）。
> δ + 軌道情報でn_wsの予測が改善するか検証中。

## Phase 1 結果（2026-04-09実行）

### 2変数回帰の結果

| モデル | R² | R²_adj |
|--------|-----|--------|
| δ_val のみ | 0.07 | — |
| δ_val + N_d | **0.40** | 0.30 |
| δ_val + Z | 0.36 | 0.25 |
| δ_val + γ | **0.64** | 0.58 |
| δ_val + N_d + γ | **0.72** | 0.64 |
| N_d のみ | 0.11 | — |
| γ のみ | **0.52** | — |

### 解釈

1. **d電子スクリーニングは因子の一つ**（R²: 0.07→0.40）だが唯一ではない
2. **γ → n_ws相関（r=0.72）は円環的**: n_wsはγを予測するためにフィットされたパラメータ
3. **3変数（δ + N_d + γ）でR²=0.72**: だがγを使うのは理論的に不健全
4. δ + N_d の R²=0.40 は「半分は説明できる」レベル

Phase 2（軌道分解δ）完了。結果は**否定的**。

## Phase 2 結果（2026-04-09実行）

### 軌道分解δの相関

| 変数 | r vs n_ws^{1/3} | n |
|------|-----------------|---|
| δ_val | +0.26 | 15 |
| δ_s | +0.22 | 15 |
| δ_sp | +0.27 | 15 |
| δ_sp + frac_d回帰 | R²=0.28 | 15 |

**軌道分解しても改善しない。**

δ_spだけ取り出してもr=0.27でδ_valとほぼ同じ。
d軌道を除いてもn_ws相関は上がらない。

### 解釈

δ → n_wsのギャップは「d電子が混ざっているから」ではなく、
もっと根本的:

1. **Miedemaのn_wsはフィッティングパラメータ**であり、
   実際の電子密度と一致する保証がない
2. **ダイマーのδ（2原子系）とバルクのn_ws（固体）**は
   空間スケールが根本的に異なる
3. **δ_IPRのMulliken population依存性**が元素間比較を歪める

Phase 1のN_d 2変数モデル（R²=0.40）の方が軌道分解δ（R²=0.28）より良い。
「d電子のIPRの広がり」ではなく「d電子の数」が効いている。

### Paper 2への影響

**δ → n_wsの因果連鎖は、当初想定より間接的。**

正しい組み立て:
- δがmidpoint密度を直接予測する（r=0.89）→ **強い**
- sp/d分離がスラブで確認される（p=0.00008）→ **強い**
- δとn_wsの直接対応は弱い（r=0.26）→ **正直に報告**
- n_wsの起源としてのδは「一つの因子」であり「唯一の因子」ではない

### Paper 2の主張への影響

**当初の「δがn_wsの起源」は修正が必要。**

修正版: 「δは境界電子密度の直接的なドライバー（r=0.89）であり、
n_wsの変動の一部（~40%、d充填度を加えた場合）を説明する。
残りの変動は結晶構造、相対論効果、Miedemaのフィッティング手続き
自体の影響を含む可能性がある。」

## Phase 3 結果（2026-04-09実行）

### DFT中間点密度（絶対値）vs Miedema n_ws

| 相関 | r | p |
|------|---|---|
| n_mid(abs) vs n_ws | **+0.66** | 0.010 |
| n_mid(abs) vs n_ws^{1/3} | **+0.68** | 0.007 |
| n_bulk vs n_ws | +0.44 | 0.12 |
| **ratio vs n_ws^{1/3}** | **−0.55** | 0.04 |

### 驚きの発見

**正規化（ratio = n_mid/n_bulk）するとn_wsとの相関が負になる。**

d金属はn_bulkが大きい（d電子が大量にある）→ ratioは小さい。
しかしn_ws自体は大きい（d電子がセル境界密度に寄与する）。

つまり:
- **ratio（δが測るもの）**: sp電子の均一性を反映。sp > d。
- **絶対密度（n_wsが測るもの）**: 電子の総量を反映。d金属も高い。

### Phase 1-3 の総合像

```
δ → ratio (r=0.89)    ... δは「質」（均一性）を測る
n_mid(abs) → n_ws (r=0.66)  ... 絶対密度はn_wsと中程度に対応
ratio → n_ws (r=-0.55)  ... 正規化すると逆転

→ δが説明するのは n_ws の「質」（均一性）であり、
  「量」（絶対密度）は電子総数（d電子数）に依存
```

### Paper 2の修正版主張

**旧**: δ → n_ws → γ（δがn_wsの起源）

**新**: δは境界電子密度の**均一性**を測る（r=0.89）。
Miedemaのn_wsは境界の**絶対密度**（r=0.66）に対応し、
均一性（δ）と電子総数（d充填度等）の両方で決まる。
δはn_wsの「質的な起源の一部」であり、
「定量的な予測子」ではない。

## 残りの検証項目の評価（2026-04-09）

### 検証D: 結晶構造効果（Fe BCC vs FCC）— 保留

Phase 3でn_mid(abs) vs n_wsがr=0.66と出た。構造効果が
このギャップの原因かを確認するにはFe FCC新規スラブが必要。
ただしPhase 3の結果から、ギャップの主因は「ratio vs 絶対値」の
指標の違いであることが判明。構造効果は二次的と判断。
**優先度: 低。将来課題。**

### 検証E: 相対論効果（Ag）— 保留

AgのPySCFダイマーは非相対論的B3LYPで計算。相対論補正で
d軌道が収縮しδが変わる可能性がある。ただしAgはn_mid/n_bulk=0.37で
Cu(0.33)やZn(0.28)と同じグループに入っており、
sp/d分離の定性的議論には影響しない。
**優先度: 低。定量的精度が必要になった段階で実施。**

### 検証F: エネルギー窓感度 — 不要

Paper 2の既存ILDOS感度テスト（5金属）で±3eVの窓変化に対して
ロバストであることが確認済み。14金属への拡張は情報量が限定的。
Phase 3で発見した「ratio vs 絶対値」の問題はエネルギー窓とは無関係。
**優先度: 不要。**

## Phase 1-3 の総合結論

δ → n_ws のギャップは以下の3因子で説明される:

1. **指標の種類の違い（最大因子）**:
   δは「均一性」（正規化ratio）を測り、n_wsは「絶対密度」を測る。
   正規化するとd金属は低く出る（n_mid/n_bulk < 0.4）が、
   絶対密度ではd金属も高い（d電子が大量にあるため）。
   Phase 3のratio vs n_wsの負の相関（r=-0.55）がこれを実証。

2. **d電子数の効果（部分的因子）**:
   δ + N_d の2変数で R²=0.40（Phase 1）。
   d電子の「数」が絶対密度に寄与するため。
   ただし軌道分解δ（Phase 2）では改善しない（R²=0.28）。

3. **Miedemaのn_wsの経験的性質**:
   n_wsは「合金生成熱をフィットするパラメータ」であり、
   物理的な電子密度と完全には一致しない（r=0.66）。
   この不一致は構造効果、相対論効果、フィッティング手続き
   の影響を含む可能性がある。

**δは n_ws の「質的な起源」（なぜsp金属とd金属で違うか）を
説明するが、「定量的な予測子」ではない。
Paper 2はこの区別を正直に述べるべき。**
