# 01: Core Framework — 有効非局在化指標δ × 有効伝導次元D_effによる光学応答の現象論的統一記述

## 1. 中心主張

物質の光学応答の主要な傾向は、電子の**有効非局在化指標δ**と**有効伝導次元D_eff**の二変数で現象論的に整理できる。

**注意：本枠組みは「光学応答を完全に決定する普遍理論」ではなく、「主要な傾向を統一的に整理する現象論的フレームワーク」である。**

## 2. 有効非局在化指標δ — 操作的定義

δは「電子非局在化を表す有効指標群」であり、系に応じて最適な指標を選択・比較する。

### 2.1 候補指標群

| 指標 | 定義 | 長所 | 短所 |
|------|------|------|------|
| IPR (逆参加率) | Σ\|ψ_i\|⁴ | 直接的、計算容易 | 有限系で飽和しやすい |
| Wannier関数広がり | ⟨r²⟩ - ⟨r⟩² | 局在度の直接測定 | 計算コスト高 |
| バンド幅比 | W_band / W_atomic | 直感的 | 多バンド系で曖昧 |
| 光学的導電率 | σ(ω→0) | 実験で直接測定可能 | 温度依存性 |
| 有効質量比 | m_e / m* | バンド構造から直接 | 異方性の扱い |

### 2.2 操作的定義の方針

「δが本質変数である」と主張するのではなく、

> 上記指標群のうち、どれが光学応答の系統的整理に最も有効かを比較検証する

という立場を取る。最終的に光学応答との相関が最も高い指標をδとして採用する。

### 2.3 現段階での暫定的序列

定性的に、以下の序列は指標の選択によらず安定：

sp3局在（ダイヤモンド）< quasi-sp2（C60）< sp2-1D（CNT）≤ sp2-2D（グラフェン）< 自由電子金属

## 3. 有効伝導次元D_eff — 操作的定義

D_effは結晶学的次元でも電子状態の厳密な次元でもなく、**低エネルギー光学応答に寄与する有効伝導次元**として定義する。

### 3.1 なぜ「有効」が必要か

ダイヤモンドの電子は固体電子論的には3DのBloch状態を形成する。しかし光学応答に寄与する低エネルギー電子伝導は、バンドギャップにより事実上ゼロ。よって光学的有効伝導次元D_eff ≈ 0。

これは「電子が0次元」という主張ではなく、「光学応答に寄与する伝導チャネルの有効次元がゼロ」という操作的記述。

### 3.2 各系のD_eff

| 物質 | 結晶学的次元 | D_eff | 根拠 |
|------|-------------|-------|------|
| ダイヤモンド | 3D | ≈ 0 | 可視域で光学伝導なし |
| C60固体 | 3D (fcc) | ≈ 0 (曲面) | 分子間ホッピング弱、分子内π系は0D曲面 |
| SWCNT | 1D | 1 | 軸方向のみ伝導 |
| グラフェン | 2D | 2 | 面内Dirac伝導 |
| グラファイト | 3D (層状) | 2 (異方的) | 面内伝導 >> 面間伝導 (100:1) |
| 金属 | 3D | 3 | 等方的伝導 |
| h-BN | 2D (層状) | ≈ 0 | バンドギャップ大、光学伝導なし |

## 4. 光学応答の分類 — 二層モデル

### 4.1 上位層：光沢の普遍的発生条件
光沢（鏡面反射）の発生には以下が必要：
- 界面の存在
- 表面が波長スケールで十分平滑
- 界面での屈折率差

この条件を満たせば、金属でも液体でも研磨ガラスでも光沢は発生する。

### 4.2 下位層：δ×D_effによる質的分岐

δ×D_effが決定するのは、光沢の「強さ・質感・色味」および「透明/色/黒/光沢」の分岐。

| δ | D_eff | 光学応答 | 物理 | 例 |
|---|-------|---------|------|-----|
| 低 | ≈ 0 | 透明 | 光子が電子応答に出会わない | ダイヤモンド, NaCl, h-BN |
| 中 | ≈ 0 | 色（選択的吸収） | 特定エネルギーでのみ応答 | C60, 銅錯体 |
| 高 | 1 | カイラリティ依存 | 1D伝導チャネル | SWCNT |
| 高 | 2 | 黒 + 劈開面光沢 | 面内で応答、面間に吸収 | グラファイト |
| 高 | 3 | 金属光沢 | 全方向で強い応答 | Au, Ag, Cu |

### 4.3 因果構造

```
有効非局在化指標δ × 有効伝導次元D_eff
    ↓
低エネルギー光学応答の強度と異方性
    ↓
誘電関数ε(ω)のスペクトル形状
    ↓
反射率R(ω) / 吸収係数α(ω) / 透過率T(ω)
    ↓
光沢 / 黒 / 色 / 透明 の知覚
```

## 5. 従来理論との関係

本枠組みは既存理論を否定しない。上位の整理変数を提供する。

- **Drudeモデル**：D_eff=3, δ高 の特殊ケース
- **バンド理論**：バンドギャップはδとD_effの帰結の一つ（ただしギャップだけでは光学応答を決定できない点を本枠組みが補完）
- **配位子場理論**：10Dqパラメータ ≈ 局所的δの変調
- **Fresnel光学**：上位層（界面反射条件）の記述。下位層（反射率の強度）にδ×D_effが効く

**本枠組みが埋める穴：** 「なぜ同じバンドギャップゼロでも金属は光沢でグラファイトは黒いのか」に統一的回答を与える。既存のバンドギャップ→光学応答の不完全な写像に、δ×D_effという変数を追加して完全化を試みる。

## 6. 第2補正の必要性

δ×D_effは第1近似。以下の場合に第2補正が必要：

- **極性/イオン性（パラメータp）**：h-BNの白さはδだけでなくB-N間の電気陰性度差（イオン性）も寄与。h-BNとグラファイトの差をδのみで押し切ると不十分な可能性。
- **励起子効果**：バンドギャップ近傍での束縛励起子が吸収スペクトルを修正
- **多体効果**：強相関系ではδの単粒子的定義が不適切になる可能性

これらを明示的に認めることで、枠組みの射程と限界が明確になる。
