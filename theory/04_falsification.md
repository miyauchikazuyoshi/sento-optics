# 04: 反証条件（修正版）

## 主論文スコープ内の反証条件

### R1: 炭素同素体の系統性反証
炭素同素体（ダイヤモンド, C60, SWCNT, グラフェン, グラファイト）の光学特性がδ×D_effの二変数で系統的に整理できない場合、枠組みの核心が棄却される。

**具体的テスト**：
- δ（操作的定義の複数候補）とバンドギャップの逆相関が全指標で |r| < 0.5
- δ×D_effによる光学応答カテゴリ（透明/色/黒/光沢）の分類正答率 < 80%

### R2: h-BN対照系反証
h-BN（D_eff ≈ グラファイト、δ < グラファイト）がグラファイトと同様の黒さを示す場合、「D_effだけでなくδも独立変数」という主張が棄却される。
→ **現状：h-BNは白い。この条件は生存。**

### R3: グラフェン積層数反証
グラフェンの積層数Nと光学吸収率が、πα×Nの線形関係から系統的に逸脱する場合、「2D壁の積層」モデルが棄却される。

### R4: δ-反射率相関反証
δの異なる系で、反射率との正の相関が得られない場合、「δが光学応答の有効指標である」という主張が棄却される。

### R5: δ定義の一貫性反証
複数のδ候補指標（IPR, Wannier, バンド幅比等）が光学応答との相関で全く異なる序列を与える場合、「δという有効指標が存在する」という前提自体が危うくなる。

## 拡張スコープの反証条件（別稿用）

### R6: 相転移接続の反証
Lindemannパラメータと光学応答の相関が複数物質系で成立しない場合。

### R7: 周期表方向の反証
同一周期内の金属→非金属遷移がδの連続的減少として記述できない場合。

## 反証を試みて生存した既存事実

| 事実 | 反証の試み | 結果 |
|------|-----------|------|
| 全液体が光沢を持つ | 光沢のない液体を探す | 反例なし（生存） |
| ざらざら金属が色味を保つ | 表面幾何だけで光沢を説明 | δ×D_effの二成分分離で説明可能（生存） |
| グラファイトの黒さと劈開面光沢の共存 | D_effだけで説明 | D_eff=2の異方性として説明可能（生存） |
| 金属もグラファイトもギャップゼロだが応答が異なる | ギャップだけで説明 | D_effの差（3 vs 2）で分岐（生存） |
| h-BNがグラファイトと同構造なのに白い | D_effだけで説明 | δの差で第1近似的に説明可能（生存、ただし極性の第2補正が必要な可能性） |

## 査読で突かれうる点と対応方針

| 指摘 | 対応 |
|------|------|
| δの定義が一意でない | 操作的定義として複数候補を比較検証する立場を明示 |
| D_effが恣意的 | 「光学的有効伝導次元」として操作的に定義。結晶学的次元との混同を避ける |
| h-BNの白さはδだけで説明できるか | 第1近似としてδ、第2補正として極性pを導入する用意があることを明示 |
| 既存理論で十分では | 「ギャップゼロの金属 vs グラファイト」の分岐を既存理論が統一的に説明していない点を指摘 |
| 射程が広すぎないか | 主論文は炭素+h-BNに絞る。拡張は明示的に将来課題とする |
