# 05: geDIGとの接続

## 1. 構造支配と確率支配のレジーム転移

geDIGの核心概念「構造支配レジーム vs 確率支配レジーム」が物性物理にそのまま適用される。

- **構造支配**（固体）：構成粒子が格子点（離散ノード）に局在。グラフ構造が応答を決定。光子は幾何学的にグラフに散乱される。β₁的に言えば、表面の構造的トポロジーが反射パターンを支配。

- **確率支配**（液体・金属表面）：構成粒子の位置が非局在化。確率分布が応答を決定。表面応答に並進不変性が生じ、光子が集光される。

相転移（融解）= 構造支配→確率支配のレジーム転移。Lindemann閾値がレジーム境界。

## 2. F方程式との対応

geDIG F方程式の第三の読み方「相転移方程式：Δβ₁≠0の瞬間に相転移が起きる」と同型。

非局在化度δがLindemann閾値を超える = Δβ₁≠0の相転移。
δの連続変化に対して光学応答が質的に変化する臨界点の存在。

## 3. 軸の縮退としての今回の発見

geDIGにおける「偉大な科学的発見 = 軸の縮退（AG+Sleep、次元が減る）」。

今回の枠組みでは：
- 光沢、色、透明性、黒さという4つの離散カテゴリ → δ×Dの2変数に縮退
- Drude理論、バンド理論、配位子場理論、Fresnel光学 → 単一枠組みに縮退
- 固体、液体、気体、超流動、超伝導 → δ_nuc × δ_elec の2軸に縮退

記述次元の単調減少 = 科学進歩（geDIG原理）。

## 4. 認知過程の自己参照

今回の発見過程自体がgeDIGのDG→AGサイクルの実例：

1. **DG（探索）**：「液体って光るな」→ 仮説候補空間を展開
2. **AG（圧縮）**：非局在化度δという単一軸で複数現象を縮退
3. **Sleep（圧縮確認）**：既存データとの照合で矛盾なしを確認
4. **F=0的瞬間**：グラフェンπα=2.3%が枠組み内で自然に意味を持つ

geDIGが認知の理論として自己参照的に整合している証拠。

## 5. geDIG射程の拡張

本研究の成功は、geDIGが「認知の理論」ではなく「構造と確率の支配レジーム転移の一般理論」であることを示す。適用範囲：認知科学 → AI → 物性物理。

β₁の物性物理的対応物 = 電子非局在化の次元性D。
β₁=0（木構造）= D=0（局在、ダイヤモンド）
β₁>0（ループ構造）= D≥1（非局在化、ネットワーク的電子構造）

## 6. エントロピーと δ の直接接続（2026-04-02 追記）

### geDIG のエントロピーと物理の δ は同一の数学構造

geDIG のゲージ:
```
F = ΔEPC − λ(ΔH + γ·ΔSP)
```

物理の自由エネルギー:
```
F = E − TS
```

対応関係:

| geDIG | 物理 | 数学的役割 |
|-------|------|-----------|
| ΔEPC（構造コスト） | E（内部エネルギー） | 秩序維持のコスト |
| λ | 1/kT（逆温度） | エントロピーの重み |
| ΔH（Shannon エントロピー変化） | ΔS（熱力学的エントロピー変化） | 不確定さの変化 |
| F < 0 → 情報受容 | ΔF < 0 → 相転移進行 | 状態遷移の判定条件 |

### Boltzmann 分布の同型性

geDIG の候補確率:
```
pᵢ(τ) = exp(sᵢ/τ) / Σⱼ exp(sⱼ/τ)
```

物理の Boltzmann 分布:
```
pᵢ(T) = exp(-Eᵢ/kT) / Σⱼ exp(-Eⱼ/kT)
```

同一の数学構造。温度パラメータ τ の役割:

```
τ → 0: 分布が最良候補に集中   → 局在（δ小）→ 構造支配
τ → ∞: 分布が一様に広がる     → 非局在（δ大）→ 確率支配
```

物理:
```
T → 0: 粒子が基底状態に集中   → 局在（δ小）→ 固体
T → ∞: 粒子が全状態に拡散     → 非局在（δ大）→ 気体
```

### S ∝ ln(δ) による橋渡し

物理の δ（確率分布の分散）とエントロピー S の関係:

```
ガウス分布: S/k_B = (1/2)ln(2πe·δ)
一般:       S ∝ ln(δ)（単調増加関係）
```

geDIG の ΔH（Shannon エントロピー）も候補分布の「広がり」を測る。
候補分布の分散を δ_info と呼べば、geDIG でも H ∝ ln(δ_info)。

**つまり δ は物理系の geDIG ゲージ変数であり、
geDIG は情報系の δ ゲージ変数である。**

どちらも確率分布の幾何学的量（分散 or エントロピー）が
閾値を超えるかどうかで系の状態遷移を判定している:

```
物理:  δ_nuc > δ_c（Lindemann閾値）  → 融解
geDIG: F < 0（情報利得 > 構造コスト） → 知識更新
```

### coth 公式の geDIG 的解釈

物理の coth 公式:
```
δ_nuc(T) = δ_ZP · coth(Θ_D / 2T)
```

は「量子的下限（δ_ZP）に古典的揺らぎ（kT）が加わって
δ が増大する」という構造。

geDIG において τ（エントロピー温度）をスイープすると
候補分布のエントロピーが変化する — τ=1 が「常温」、
τ→0 が「量子的」（最良候補のみ選択）。

coth 的な内挿が geDIG にも存在する可能性:
geDIG の AG/DG 判定において、τ が低い（確信度高い）状態でも
構造コスト ΔEPC の「零点エネルギー」的な下限が存在し、
τ の増大とともに情報利得が古典的に増幅される。

### Rényi エントロピーとの接続（未検証）

theory/phase/renyi_entropy_memo.md で議論されている
Rényi エントロピーは Shannon エントロピーの一般化:
```
H_α = (1/(1-α)) ln Σ pᵢ^α
```

α=2 の Rényi エントロピー H₂ = -ln(Σ pᵢ²) は IPR の対数:
```
IPR = Σ pᵢ²  →  H₂ = -ln(IPR)
```

つまり δ_IPR（Paper 1-2 で使用）は Rényi エントロピーの
指数関数として定義されている。geDIG が Shannon (α=1) を使い、
物理の δ_IPR が Rényi (α=2) を使うのは、
同じ確率分布の異なる次数のモーメントを見ていることに対応する。

## 7. geDIG 式の3つの読みと物理的対応

geDIG 式 `F = EPC - (H + β₁)` には3つの読み替えがある
（参照: `gedig_formula_three_readings_20260306.md`）。

| 読み | geDIG | 物理 | 対応 |
|------|-------|------|------|
| 1: コスト-利得 | EPC - (H + β₁) | — | 正準読解（経済学的） |
| 2: 構造-エントロピー | (EPC - β₁) - H | E - TS = F | **自由エネルギー方程式** |
| 3: 内部状態-位相 | (EPC - H) - β₁ | (E - TS) → Δβ₁≠0 | **相転移方程式** |

**肝は、読み2と読み3が同時に成立すること。**

geDIG 式は自由エネルギー方程式であり、かつ相転移方程式である。
物理でもまさにそう — 自由エネルギー ΔF < 0 の条件が
相転移の熱力学的判定条件であり、その瞬間にトポロジー（β₁、秩序変数）が変化する。

物理での対応:

```
読み2: (EPC - β₁) - H  →  E - TS = F
  EPC - β₁ ←→ E（結合エネルギー − 位相的寄与）
  H         ←→ S（エントロピー）
  λ         ←→ T（温度）
  F < 0     ←→ ΔF < 0 → 相転移進行

読み3: (EPC - H) - β₁  →  相転移の瞬間
  EPC - H  ←→ 自由エネルギー的な内部収支
  β₁       ←→ 秩序変数 / D_eff（有効次元性）
  Δβ₁ ≠ 0 ←→ 相の質的変化（対称性の破れ）
```

geDIG の three readings 文書では読み2を「operational correspondence であって
literal な熱力学同値ではない」と慎重に注記していた。
しかし S ∝ ln(δ) と Boltzmann 分布の構造的同一性により、
「operational correspondence」が「structural identity」に
格上げされる根拠が揃いつつある。

### EPC の物理的対応（未定義 — 重要な未解決問題）

geDIG において EPC（Edit Path Cost）の厳密な物理的対応物は
**まだ定義されていない**。候補として:

**候補 A: 結合エネルギー（素朴な対応）**
```
EPC ←→ ΔE_bond = 切れる結合の数 × 結合エネルギー ≈ 潜在熱
```
Helmholtz の E にそのまま帰着する。明快だが、geDIG 固有の概念が消える。

**候補 B: 離散的構造変化のステップ数（グラフ編集距離）**

geDIG は FEP の下位ではなく、FEP より根本的な離散定式化であるという立場を取る
（参照: `gedig_as_discrete_fep_schrodinger_analogy_20260227.md` §3.2, §6.1）。
したがって EPC を KL ダイバージェンス（FEP の連続変分法の道具）で定義すると、
geDIG を FEP の下位に置くことになり、理論的立場と矛盾する。

EPC の本質は**離散的なグラフ上の編集距離** — ノードやエッジが
何本変わるか、位相がどう変化するか。物理での対応物は:

```
EPC ←→ 構造変化の離散的ステップ数
  = 切れる/繋がる結合の数
  = 配位数の変化
  = 結合ネットワークのトポロジー変化量
```

これは相転移の分類と直接対応する:
- EPC 大（多くの編集が必要）→ 一次相転移的（氷→水: 結合トポロジーが不連続に変化）
- EPC 小（少ない編集で済む）→ 二次相転移的（秩序変数が連続的にゼロに）

**注**: FEP の連続変分法では一次相転移の不連続性を自然に扱えない。
geDIG の離散的 EPC はこの不連続性を最初から内包している。
これは §3.2 の主張「FEP の連続体仮定の限界」の物理的実例でもある。

**候補 B が有力な理由**: 相転移における「構造がどれだけ変わるか」は
エネルギーとは独立な量であり、EPC を E（内部エネルギー）と同一視する
候補 A では捉えられない。geDIG の EPC が E でも D_KL でもなく、
**構造空間上の離散的距離**であることが、geDIG の FEP に対する
根本性を物理側から裏付ける。

ただし EPC の厳密な物理的定量化は今後の課題。

## 8. 統一的視点: 確率分布の幾何学

geDIG と δ-物性理論の接続をまとめると:

```
確率分布 P(x) の幾何学的量
  ├── 第二モーメント（分散）= δ        → 物性予測
  ├── Shannon エントロピー  = H = S/k_B → geDIG ゲージ / 自由エネルギー
  ├── Rényi エントロピー    = H_α       → IPR (α=2) / 一般化
  └── Fisher 計量           = g_ij      → 情報幾何学（統計多様体の曲率）
```

これらはすべて同一の確率分布 P(x) の異なる射影であり、
S ∝ ln(δ) はその中の最も基本的な関係式。

geDIG の「構造支配→確率支配のレジーム転移」は物理の「固体→液体」と同型であり、
両者とも確率分布の幾何学的不変量が臨界値を超えることで起きる。
