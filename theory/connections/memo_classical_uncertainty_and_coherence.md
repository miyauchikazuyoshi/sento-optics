# Memo: 古典的不確定性としての δ_nuc とコヒーレンス条件

**Status: 直感メモ — 未検証、論文未反映**
**Date: 2026-04-01 (updated 2026-04-02)**

## 1. 核心的直感

> 液体の粒子の「居場所」は原理的に特定できない。
> これは量子的な不確定性ではなく、熱揺らぎによる「古典的不確定性」である。
> この非局在度がある閾値を超えるとき、系のコヒーレント応答が変質する可能性がある。

---

## 2. 「水分子の居場所が特定できない」の物理的裏付け

液体水では構造的に居場所が曖昧になる条件が成立している：

```
水の酸素原子の平均自由行程：~ 2-3 Å
水分子の直径：             ~ 2.8 Å

移動距離 ≈ 粒子サイズ  
→ 「どこにいるか」が構造的に不明確
```

中性子散乱による動径分布関数でも、液体水は結晶水と比べて  
非常にブロードであり、「居場所」の非局在性が直接観測されている。

---

## 3. 量子的不確定性との対応

```
量子の不確定性：
  ΔxΔp ≥ ℏ/2
  原因：波動関数の本質的広がり
  スケール：ℏ

古典（熱）の不確定性：
  ΔxΔp ~ kT
  原因：熱揺らぎによる位相空間の占有
  スケール：kT

液体水（室温）：
  Δx ~ 2-3 Å（構造的揺らぎ）
  Δp ~ √(2mkT)（熱運動）
  ΔxΔp ~ kT >> ℏ/2  ← 古典的だが「不確定」
```

**ℏ/2 と kT は同じ次元・同じ形をしている。スケールが違うだけ。**

不確定性原理の古典版が熱力学的な揺らぎである、という対応。

### 先行研究（この対応は既知）

量子-古典の不確定性積クロスオーバーは、調和振動子で厳密に定式化されている:

```
ΔxΔp = (ℏ/2) · coth(ℏω/2kT)

  低温 (kT << ℏω): → ℏ/2     （量子的下限）
  高温 (kT >> ℏω): → kT/ω    （古典的熱揺らぎ）
  クロスオーバー:    kT ~ ℏω
```

関連する主要文献:

| 論文 | 内容 | δ との関係 |
|------|------|-----------|
| Usha Devi & Karthik (2012) Am. J. Phys. 80, 708 | 古典アンサンブルの不確定性積が量子と同一の数学構造 | δ_nuc の古典-量子対応の数学的基盤 |
| Mandelbrot (1956) IRE Trans. IT-2, 190 | ΔU·Δ(1/T) ≥ k_B（熱力学的不確定性関係） | k_B が ℏ の役割を果たす対応 |
| Uffink & van Lith (1999) Found. Phys. 29, 655 | Mandelbrot 関係は QM と厳密に類似、ただし相補性は否定 | 「古典的不確定性原理」の限界の明確化 |
| Zhu & Wang (2024) arXiv:2407.06715 | Δp ≥ √(2π)ℏ/λ_th（ボルツマン下限） | 熱揺らぎが ℏ/2 より厳しい下限を課す |

**新規性の所在**: これらの先行研究はいずれも不確定性積の数学的構造を議論しているが、
「不確定性→非局在度δ→物性予測（光学応答・表面張力）」という接続は行っていない。
coth 公式は δ_elec (Ω) と δ_nuc を統一的に扱う数学的基盤として利用できる可能性がある。

## 4. δ_nuc の飽和条件の再定式化

```
量子系の δ の下限：ΔxΔp ≥ ℏ/2 で決まる
古典系の δ の下限：ΔxΔp ~ kT で「実効的な下限」が決まる
```

これを δ_nuc の言葉で書くと：

```
液体の条件：δ_nuc ~ δ_particle

粒子の非局在度（熱揺らぎによる広がり）  
  ≈ 粒子自身のサイズ
```

固体では δ_nuc << δ_particle（格子振動の振幅 << 粒子サイズ）。  
気体では δ_nuc >> δ_particle（平均自由行程 >> 粒子サイズ）。  
液体はその境界：δ_nuc ≈ δ_particle。

これは Lindemann 融解基準（√⟨u²⟩/d ~ 0.1）の  
δ 言語による自然な再表現でもある。

## 5. コヒーレンス保存の閾値条件（検討中）

電子の非局在度 Ω が大きいと、隣接原子の電子雲が重なり合い、
入射光に対して集団応答が可能になる。
Cardenas-Castillo sum rule (Ω_I = ∫ε₂/ω dω) はこれを定量化している。

核でも同様に、δ_nuc がある閾値を超えると
「個別の散乱体」という描像が壊れ、連続体として振る舞う可能性がある。

```
δ_nuc << d（格子間距離）：
  各粒子が独立した散乱体 → インコヒーレント散乱

δ_nuc ~ d：
  粒子位置の不確定性が格子間距離と同程度
  → 「離散的な散乱点」の描像が崩壊
  → 連続体近似が成立 → コヒーレント応答が可能に
```

閾値は Lindemann 条件（√⟨u²⟩/d ~ 0.1）と整合する。
ただし、古典液体では λ_dB << d であり、
このコヒーレンスが量子的起源か古典的（統計的平均化）起源かは
慎重に区別する必要がある。
