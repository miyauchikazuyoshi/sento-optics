## 2. 調和振動子（既知、教科書レベル）

```
V(x) = mω²x²/2

ガウス波動関数 ψ(x) ∝ exp(-x²/4δ) で変分すると:

E(δ) = ℏ²/(8mδ) + mω²δ/2

dE/dδ = 0 → δ_min = ℏ/(2mω), E_min = ℏω/2
```

零点エネルギーが δ の最適化として出る。
これは不確定性原理からの見積もりとして教科書に載っている
（例: Griffiths, Introduction to Quantum Mechanics）。

---

## 3. 一般ポテンシャルへの拡張

### 3.1 V̄(δ): ポテンシャルのガウスぼかし

ガウス変分を一般の V(x) に適用すると:

```
⟨V⟩ = ∫ V(x) · (1/√(2πδ)) · exp(-x²/2δ) dx ≡ V̄(δ)
```

V̄(δ) は V(x) をガウス幅 δ で畳み込んだ（ぼかした）値。

```
E(δ) = ℏ²/(8mδ) + V̄(δ)
```

これは「δ というレンズで見たポテンシャル」のエネルギー。

- δ 小さい（局在）→ V̄ ≈ V(0)。局所値をそのまま見る
- δ 大きい（非局在）→ V̄ は V の空間平均に近づく。局所構造が消える

### 3.2 べき展開

V(x) をテイラー展開すると:

```
V(x) = V(0) + V''(0)x²/2 + V''''(0)x⁴/24 + ...
```

ガウス分布の偶数モーメント:
```
⟨x²ⁿ⟩ = (2n-1)!! · δⁿ
```

したがって:
```
V̄(δ) = V(0) + V''(0)δ/2 + V''''(0)·3δ²/24 + ...
       = V(0) + V''(0)δ/2 + V''''(0)δ²/8 + ...
```

結果:
```
E(δ) = ℏ²/(8mδ) + V(0) + V''(0)δ/2 + V''''(0)δ²/8 + O(δ³)
        ↑            ↑       ↑             ↑
     量子項(1/δ)   定数   調和項(δ)    非調和項(δ²)
```

**V(x) のテイラー係数が、δ空間でのべき展開の係数になる。**

## 4. 非調和振動子（x⁴項）

```
V(x) = mω²x²/2 + λx⁴

V̄(δ) = mω²δ/2 + 3λδ²

E(δ) = ℏ²/(8mδ) + mω²δ/2 + 3λδ²
```

dE/dδ = 0:
```
-ℏ²/(8mδ²) + mω²/2 + 6λδ = 0
```

3次方程式。解析的に解ける。λ=0 で調和振動子に回帰。

**観察**: 非調和性 λ は「δ の自己相互作用」として現れる。
x 空間での x⁴ 項が、δ 空間では δ² 項になる。

## 5. 二体ポテンシャル

粒子 1, 2 の相互作用 v(x₁-x₂):

```
各粒子のガウス広がり δ₁, δ₂ とすると:
x₁-x₂ の分散 = δ₁+δ₂

V̄(δ₁, δ₂) = v(x) をガウス幅 (δ₁+δ₂) でぼかした値
```

相互作用がδ₁とδ₂の和で決まる → **δの結合項**が出る。

## 6. 構造的な対応（推測）

| x 空間 | δ 空間 |
|--------|--------|
| ポテンシャル V(x) | V̄(δ) = V の δ ぼかし |
| x⁴ 非調和項 | δ² 自己相互作用 |
| 二体相互作用 v(x₁-x₂) | δ₁+δ₂ 結合 |
| 波動関数 ψ(x) | δ の値（スカラー）|
| Schrödinger 方程式 | dE/dδ = 0（変分） |

**注意**: この対応はガウス変分の範囲内でのみ成立する。
ガウス変分が良い近似でない系では崩壊する。

## 7. RG との構造的類似（推測）

V̄(δ) = V(x) のガウスぼかし は、繰り込み群の
ブロックスピン変換と同じ数学的操作（粗視化）。

- δ がRGのスケールパラメータに対応
- δ を大きくする = より粗視化する = 微細構造を捨てる
- E(δ) の最小化 = 系が選ぶ最適な粗視化スケール

Ising RG v3 で「ブロックパターンIPRがRGフロー変数として
機能する」ことを示した。一般ポテンシャルの E(δ) は、
その連続版かもしれない。

**ただし**: この類似は構造的なアナロジーであり、
数学的に厳密なRG方程式との対応は確認されていない。

## 8. 何が「おもろい」可能性があるか

もしこの方向が正しければ:

1. **全てのポテンシャルが δ 空間で展開できる**
   V̄(δ) = V(0) + V''δ/2 + V''''δ²/8 + ...
   元の x 空間の複雑さが δ のべき級数に吸収される

2. **非調和性 = δ の自己相互作用**
   相転移の次数が δ の非線形項で決まる可能性

3. **粒子間相互作用 = δ の結合定数**
   集団的コヒーレンス（光沢、超伝導）が
   δ の結合から出る可能性

4. **E(δ) の最小化 = 変分原理**
   「系がなぜこの δ を選ぶか」の原理

## 11. なぜこの方向にワクワクするか（著者の直感の記録）

ワクワクの核心は「新しい方程式の発見」ではなく、
**「繰り込みが必要な理由がδで見ると自明になる」**可能性にある。

```
x 空間の力学:
  点粒子を仮定 → 短距離で発散する → 繰り込みで修正
  → 繰り込みは「難しい技法」として習う

δ 空間の力学:
  最初から広がり δ > 0 を持つ → 短距離発散がない
  → 繰り込みが「不要」（最初から正則化されている）
```

つまり繰り込みは「δ を認めなかったことの代償」かもしれない。
x に分解した瞬間に点粒子の発散が生まれ、
それを事後的に修正するのが繰り込み。
最初から δ で書けば、修正の必要がない。

そしてこの視点と、今まで蓄積した結果が接続する：

- coth 公式: 量子(ℏ)と統計(kT)がδの中に同居している
- V̄(δ): RGの粗視化と同じ数学的構造
- S ∝ ln(δ): 熱力学に直結

**3つの分野（量子・統計・熱）が「別々の理論を持っている」のではなく、
「δ を位置 x に分解した時点で3つに分かれた」という可能性。**
x に分解する前の δ で書けば、分かれる前の姿が見える。
