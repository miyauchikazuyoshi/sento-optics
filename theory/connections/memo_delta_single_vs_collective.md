# δの階層構造: 1粒子δ vs 系のδ、そして液面コヒーレンスの条件

**Status: 2026-04-04。概念操作メモ。完全に未検証。**

## 1. δは2つのレベルで定義できる

### 1粒子δ

波動関数ψ_iの空間的広がり。

- 電子: Wannier spread Ω_i = ⟨r²⟩_i - ⟨r⟩_i²
- 原子核: 零点振動 ⟨u²⟩_i（固体）、MSD ⟨r²(t)⟩_i（液体）

これは粒子 i の性質。

### 系全体のδ

N粒子系の集団的な非局在化。

- 独立粒子近似: δ_sys = (1/N) Σ_i δ_i（一体IPRの平均）
- 相互作用あり: δ_sys ≠ (1/N) Σ_i δ_i。差分が電子相関の寄与。
- Paper 2 のδ_IPR はKS軌道の平均 → 独立粒子近似に対応。

## 2. 関係式（未検証の枠組み）

### 独立粒子（相互作用なし）

```
δ_sys = (1/N) Σ_i δ_i
```

フェルミオンの場合、Pauli排他律でSlater行列式が強制されるが、
軌道が直交していれば1粒子δの平均で系のδが決まる。

### 相関がある場合

多体波動関数Ψが単一Slater行列式で書けない。
自然軌道の占有数 n_i が 0 と 1 の間の値を取る（部分占有）。

```
δ_sys = F(δ_1, ..., δ_N, {n_i}, {⟨δ_i δ_j⟩})
```

2体相関 ⟨δ_i δ_j⟩ が入る。粒子間のδの「結合」が
系全体のδを変える。

### 相転移 = 1粒子δは変わらないのに系のδが劇的に変わる

| 相転移 | 1粒子δ | 系δ | 何が起きるか |
|--------|-------|------|-----------|
| 融解 | δ_nuc,i ≈ 一定（零点振動） | δ_nuc,sys: 0 → 有限 | 個別振動 → 集団拡散 |
| 超伝導 | δ_elec,i ≈ 一定（Bloch） | δ_elec,sys → MAX | Cooper対凝縮 |
| BEC | δ_i: 有限 | δ_sys → MAX | 全粒子が単一状態に |

**相転移 = δの「個→集団」への創発。**

## 3. 微分可能性

δは以下の変数で微分可能:

### 時間微分 ∂δ/∂t

拡散方程式から: δ_nuc = ⟨r²(t)⟩/a² → dδ_nuc/dt = 2dD/a²。
融解でdδ/dtが不連続ジャンプ、ガラス転移では連続的に0に漸近。
**相転移の次数がdδ/dtの連続性で分類できる可能性。**

### 位置微分 ∇δ(r)

界面で最も重要。バルクから表面に向かってδが変化する。
∇δが表面張力の微視的起源: 「δの勾配がある場所にエネルギーコスト」。
Cahn-Hilliard理論の|∇φ|²項と同型。
界面エネルギー ∝ ∫|∇δ|²dV が自然に出る。

### エントロピー微分 ∂δ/∂S

δ_IPR = e^{-H₂} なのでδとSは対数関係。
∂δ_nuc/∂S > 0（エントロピー増大 → 非局在化増大）。
δがエントロピーの操作的代理変数になる。

### スケール微分 dδ/dℓ（RGのβ関数）

粗視化スケールℓでδがどう変わるか。
Ising RG v3で検証: Tcでdδ/dℓ ≈ 0（フロー停滞 = 固定点候補）。

### δの場の方程式（推測）

全部微分可能なら偏微分方程式が書ける:

```
∂δ/∂t = D_δ ∇²δ + f(δ, T)
```

δ自体の拡散方程式。Cahn-Hilliard / Ginzburg-Landau方程式と同型。
**これが物理的に意味があるか、既存の場の理論の言い換え以上かは未検証。**

## 4. 液面コヒーレンスとδの共有（著者の直感）

### 問い

液面が光沢を持つ（光子の位相コヒーレンスを保存する）ためには、
界面の粒子が**集団的に**コヒーレントな応答をする必要がある。
これは1粒子δだけでは不十分で、**粒子間でδが「共有」されている**
必要があるのではないか？

### 直感の内容

液体の個々の分子は動いている（δ_nuc,i > 0）。
しかし光沢が生じるためには、界面の分子群が
入射光子の波長スケール（~500nm）にわたって
**位相の揃った**電磁応答をする必要がある。

これは個々の分子のδ_elec,iが高いだけでは足りない。
分子間の電子雲の重なり（van der Waals的相互作用、水素結合等）を
通じて、**隣接分子のδが結合している**（⟨δ_i δ_j⟩ ≠ ⟨δ_i⟩⟨δ_j⟩）
必要がある。

### δの共有 → コヒーレンス

```
1粒子δ_i が高い → 個別の分極応答が大きい
⟨δ_i δ_j⟩ が高い → 隣接粒子の応答が相関 → 位相が揃う → コヒーレンス
```

つまり光沢の条件は:

```
光沢 = δ_i 高い × ⟨δ_i δ_j⟩ 高い × 表面平滑性
     = 1粒子の応答強度 × 粒子間のコヒーレンス × 幾何学的条件
```

Paper 1 の「光沢 = 位相保存的界面応答」を
δの言語で再記述するとこうなる。

### なぜ気体は光沢を持たないか

気体では分子間距離が大きく、電子雲の重なりがない。
→ ⟨δ_i δ_j⟩ ≈ ⟨δ_i⟩⟨δ_j⟩（無相関）
→ 各分子がバラバラに応答 → 位相がランダム → コヒーレンスなし → 光沢なし

**液体と気体の光沢の違いは、δ_i の差ではなく ⟨δ_i δ_j⟩ の差。**

これは Paper 3 の (δ_nuc, δ_elec) 相図における液体-気体区別
（δ_elec > 0 = 電子雲が重なる = 凝集力）と整合する:
δ_elec > 0 は ⟨δ_i δ_j⟩ > 0 の十分条件。

### 検証可能性

- 分子動力学シミュレーションで液体界面の
  分極応答相関関数 ⟨P_i(t) · P_j(t)⟩ を計算し、
  液体/気体で比較する
- 相関長ξ_δ（δの空間相関が減衰するスケール）を定義し、
  ξ_δ >> λ_light のときコヒーレント反射が可能

**未検証。概念操作の段階。**

## 5. Ising RG結果との接続

Ising RGで磁化IPR（系全体のδ）よりブロックパターンIPR
（局所的な数粒子δ）がRGフロー変数として良かった理由:

RGは「局所δから系δへの繰り込み」のプロセスそのもの。
だから**RGの入力変数は局所δであるべき**。
系全体のδはRGの**出力**（固定点に流れた先の値）。

```
局所δ(ℓ_0) → RGフロー → 局所δ(ℓ_1) → ... → 系δ(ℓ_macro)
```

v3のブロックパターンIPRは局所δ（2×2ブロック = 4粒子の集団δ）。
v2の磁化IPRは系全体のδ。RGフロー変数は前者。
