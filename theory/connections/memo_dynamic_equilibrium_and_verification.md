## 1. δ = 動的平衡度：統計力学に存在しない変数

### 核心の主張

> 物理学には「静的平衡の記述」と「非平衡の記述」はあるが、
> 「平衡の中でどれだけ動的か」を測る変数が定義されていない。
> δはその空白に入る。

### 動的平衡の直感

- **静的平衡** (δ低): 粒子が固定された状態で安定。固体。
- **動的平衡** (δ中〜高): 粒子が常に入れ替わりながらマクロには安定。
  液体——分子は絶えず動くが、密度・表面張力・反射率は一定。
- **平衡崩壊** (δ→拘束外れ): 気体。動的平衡が崩れる。

δ = 0 は完全に静的な理想固体（零点振動を除く）。
δが上がるにつれてミクロの動きが増しながらマクロの安定性は維持される。

### 既存の「近い量」とその不足

| 既存量 | 何を測るか | 不足 |
|--------|-----------|------|
| Lindemannパラメータ | 格子振幅/原子間距離 | 融解基準のみ。連続変数として物性を予測しない |
| 拡散係数 D | 粒子の動き回り方 | 輸送係数。平衡状態の記述子ではない |
| Debye-Waller因子 | 結晶中の原子変位 | 固体に限定 |
| エルゴード性指標 | 時間平均と集団平均の一致 | 物性との直接的接続がない |

どれも「動的平衡度」の断片を捉えるが、全相・全自由度の統一的記述にはならない。

### なぜ物理学にこの変数がなかったのか

平衡統計力学の基本的立場は「平衡状態ではミクロな動きの詳細は無関係で、
分配関数 Z = Σ exp(-βE) から全て出る」。

- 粒子がどれだけ動き回っているかは、Zの中に暗黙に含まれるが
  独立変数として抽出されていない。
- 非平衡統計力学はエントロピー生成率や揺動散逸定理を扱うが、
  「平衡からどれだけ離れているか」の指標であって
  「平衡の中でどれだけ動的か」の指標ではない。

**δはこの穴に入る。** T, P, n, そしてδ。

## 2. δの「便利さ」の根拠

### 一つの定義で全スケールに適用可能

δの定義は「確率分布の一様性」。IPR = Σpᵢ² を計算して逆数を取る。
この操作は対象を選ばない。

- 電子の波動関数 → δ_elec
- 核の位置分布 → δ_nuc
- イオンの空間分布 → δ_ion
- 分子の配向分布 → δ(回転)
- 認知グラフのノード訪問確率 → geDIGのH

### 既存物理量との変換

| 既存の物理量 | δ表現 |
|-------------|-------|
| 拡散係数 D | δ_nuc の動的表現 |
| 導電率 σ | δ_elec の輸送的表現 |
| Landau-de Gennes S | 1 − δ_nuc(回転) |
| Miedema n_ws | δ_IPR の帰結 (Paper 2) |
| 粘性 η | 1/δ_nuc の抵抗的表現 |
| 反射率 R | δ_elec × D_eff の光学的表現 |
| 表面張力 γ | 界面での δ勾配 の力学的表現 |

既存の物理量は全て δの異なる射影 として再解釈できる。
δは射影される前の元の変数。

### 測定方法が複数ある

- δ_elec: IPR（計算）、導電率（測定）、反射率（測定）
- δ_nuc: 拡散係数（測定）、粘性（測定）、Lindemannパラメータ（計算）

一つの量を複数の独立な方法で測定できることは、
その量が実在する強い傍証。Paper Iの三プロキシが同じ序列（r = 0.73-0.89）はまさにこれ。

### ロゼッタストーン

異分野の現象を同じ言語で議論できる。
光学の人と冶金の人と液晶の人と輸送理論の人が、δという共通語で話せる。

## 3. 検証戦略：一入力三出力

### 構造

```
入力:  結晶構造（既知）→ Kohn-Sham軌道（DFT）→ IPR → δ

出力1: δ → n_ws → γ              (Miedema経由、Paper 2)
出力2: δ × D_eff → ε(ω) → R(ω) + n(ω)  (光学応答、Paper 1拡張)
出力3: δ → σ(ω)                   (光学伝導度)

照合:  計算された R, σ, n を既存の測定値と比較
```

一つの入力（δ）から三つの独立な出力が出て、
三つとも測定値と一致すれば、δが物性の基礎変数であることの実証になる。
一つでも系統的に外れれば、枠組みの限界が見える。
**どちらに転んでも科学として価値がある。**

### 必要なものは全て揃っている

- DFTコード: PySCF, Quantum ESPRESSO は手元にある
- 測定データ: 文献にある
- 新しい実験: 不要。計算だけで検証が完結する

## 4. geDIG H ↔ δ の数学的同型

### 対応表

| | geDIG | 物性 |
|---|-------|------|
| 基本変数 | H (認知グラフ上のノード訪問確率の不確実性) | δ (物質中の粒子の存在確率の不確実性) |
| 数学的正体 | Rényi的エントロピー | Rényi-2エントロピーの単調変換 |
| 高い場合 | 探索的 | 動的平衡（液体） |
| 低い場合 | 搾取的 | 静的平衡（固体） |
| 意味 | 構成要素がどれだけ確率的に振る舞っているか | 同左 |

### β₁の対応

- geDIG: Δβ₁ ≠ 0 で相転移
- 物性: β₁（電子的にアクティブなループ数）の変化 → 光学応答の質的変化
  - ダイヤモンド (β₁ = 0, 透明) → グラファイト (β₁ > 0, 黒+光沢)

### F方程式の物性適用

geDIG F = EPC − λ(ΔH + γ·Δβ₁)

> geDIGの方程式は認知に限定された式ではなく、
> 「群としての動的平衡系が自己組織化する一般原理」かもしれない。

認知グラフも物質も、構成要素の確率的振る舞い（δ/H）と
トポロジカル構造（β₁）と変換コスト（EPC）の三項で記述される系。

## 5. 量子-古典境界の再定義（拡張）

> 量子-古典の壁は、スケール絶対ではなく、
> コヒーレント性の相対性によって成立する。

### 従来の暗黙の前提

小さいもの = 量子的、大きいもの = 古典的。ℏが境界を定める。

### 反例

- 超伝導体: cmスケールで量子コヒーレント
- 超流動He: バケツのスケールで量子効果
- 室温単一原子: 環境結合で瞬時にデコヒア → 事実上古典的

スケールが境界を決めていない。

### δの枠組み

- δ高 → 構成要素が協調的に非局在化 → コヒーレンス保存 → 量子的
- δ低 → 構成要素が局在 → コヒーレンス散逸 → 古典的
- 同じ物質でもT, P, 環境結合によりδが変わり、量子的にも古典的にもなる

**境界はδの閾値であって、サイズではない。**

### 金属光沢 = 日常的証拠

金属表面は巨視的だが、δ_elec が高いから光子のコヒーレンスが保存 → 鏡面反射。
同じ金属を粉末にすると表面のδの空間的均一性が壊れ、光沢が消えて黒くなる。

スケールは変わっていない。変わったのはコヒーレンス保存条件。
**これは量子デコヒーレンスの光学版。**

### 定量化の方向

τ = g(δ, T, 環境結合) が導出できれば、
量子-古典クロスオーバーの定量的記述になる。

## 7. 到達地点の整理

δは以下の全てであり、同じ量の異なる側面：

1. **非局在化度** (定義)
2. **Rényi-2エントロピーの単調変換** (数学)
3. **位相コヒーレンス保存度** (光学)
4. **動的平衡度** (統計力学)
5. **群としての協調的確率化の指標** (geDIG)

一つの変数にこれだけの顔があること自体が、
基礎変数である証拠かもしれないし、過剰解釈かもしれない。
