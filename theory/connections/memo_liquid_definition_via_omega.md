# 液体のミクロ定義: Ω (Wannier spread) + 拡散方程式

## 核心的洞察 (2026-04-01)

液体状態は、電子と原子核の**両方**が「非局在化」した状態として定義できる。

### 電子の非局在化: Ω (Wannier spread)

Marzari & Vanderbilt (1997) の定義:

```
Ω_n = ⟨w_n|r²|w_n⟩ - ⟨w_n|r|w_n⟩²
```

波動関数の空間的広がり（分散）。定常的な量。単位: Å²。
Cardenas-Castillo (2024) sum rule により光学応答と直結:

```
Ω_I = (ℏ²e²/2πm²) ∫₀^∞ Im[ε(ω)] / ω dω
```

### 原子核の非局在化: 拡散方程式

Fick の第2法則:

```
∂c/∂t = D ∇²c
```

解（自由拡散）:

```
c(r,t) = 1/(4πDt)^(3/2) exp(-r²/4Dt)
```

Einstein の関係:

```
⟨r²(t)⟩ = 6Dt  (3D)
```

### 数学的対応

| | 電子 (量子) | 原子核 (統計) |
|--|------------|-------------|
| 量 | Ω = ⟨r²⟩ - ⟨r⟩² | ⟨r²(t)⟩ = 6Dt |
| 本質 | 確率分布の2次モーメント | 確率分布の2次モーメント |
| 固体 | Ω 小（局在 WF） | ⟨u²⟩ 有界（格子振動） |
| 液体 | Ω 大（広がった WF） | ⟨r²(t)⟩ ∝ t（拡散、発散） |
| 時間依存性 | 定常 | 時間とともに増大 |

**共通の数学構造**: どちらも「粒子がどれだけの空間を占有するか」の測定。
量子的 vs 統計的という起源の違いがあるが、概念は統一されている。

### 液体の定義（提案）

液体 = 以下の2条件を同時に満たす状態:
1. δ_elec = Ω/V^(2/3) が結晶に比べて増大（電子の非局在化）
2. δ_nuc: ⟨r²(t)⟩ が t に比例して発散（原子核の拡散的非局在化）

固体: δ_elec 小、δ_nuc 有界
気体: δ_nuc 発散（弾道的、∝ t²）、δ_elec → 原子軌道に縮退

## 文献状況

- Ω で液体を定義した論文は**ゼロ**（2026-04-01時点の調査）
- MLWF は液体 AIMD で使用されている（Silvestrelli & Parrinello 1999）
- 非晶質 Si で Ω 増大が観測されている（Silvestrelli et al. 1998）
- 拡散と Ω を統一的に「非局在化指標」として扱う視点は未踏

## 実験設計: 液体金属 vs 通常液体の比較

液体金属と通常液体では δ_elec と δ_nuc の寄与が質的に異なる:

| | 液体金属 (Hg, Ga) | 通常液体 (H₂O, EtOH) |
|--|-------------------|---------------------|
| δ_elec | 大（自由電子的） | 小（分子軌道に局在） |
| δ_nuc | 拡散的（原子） | 拡散的（分子） |
| 表面張力 γ | 高 (~500 mN/m) | 低 (~20-70 mN/m) |
| 屈折率 n | 金属的（複素） | 透明（実数） |
| 導電率 σ | 高 (~10⁶ S/m) | 低（イオン伝導のみ） |

**予測**: γ, n, σ の違いは δ_elec の違いで系統的に説明できるはず。
δ_nuc（拡散係数 D）は両者で同程度 → 液体としての「流動性」は共通。
物性の違いは主に δ_elec が支配。

### 具体的テスト

1. **表面張力**: γ ∝ δ_elec の相関（液体金属 vs 通常液体で2群に分離するか）
2. **光学応答**: 液体金属の ε(ω) を Ω で説明（Drude + Wannier）
3. **導電率**: σ ∝ δ_elec（液体金属 >> 通常液体）の定量的比較
4. **拡散係数 D の比較**: δ_nuc は両群で同程度かを確認（液体の共通基盤）

## 先行研究: Remsing & Klein (2020)

JPCB 124, 3180 / arXiv:2003.12158

液体Si (1800K, 216原子, CP2K/SCAN) の MLWF を計算。
電子対を spread でフィルタ分類:
- 共有結合型: σ_W < 3 Å²（固体Si Ω/WF = 3.049 と整合）
- 孤立対型: 3〜5 Å²
- 金属的/拡散型: > 5 Å²

共有結合に関与する Si は ~30%、結合寿命 ~0.6 ps。

**彼らがやっていないこと**: Ω 総量・分布の報告、固体との定量比較、
Ω を物性予測に使う枠組み、δ_elec × δ_nuc の2軸描像。
→ 全て我々の新規性として残っている。
