# 核の Resta 位相: δ_nuc の静的・物質固有な再定義（提案）

**Status: 2026-07-04。Claude (Fable 5) との議論に基づく提案。未実装・未検証。同日 Claude が [Resta & Sorella (1999)](https://arxiv.org/abs/cond-mat/9808151) を全文精読し、式を原典規約に修正、リスク1を解決（§7）、Grasselli & Baroni を要旨照合（§6）。**

## 1. 動機: 現行 MSD 定義の2つの問題

現行の δ_nuc = MSD/a²（memo_liquid_definition_via_omega.md）には:

1. **時間窓の任意性**: 液体では MSD → ∞ なので有限時間で切る必要があり、窓の選択（Frenkel 時間？ 何ステップ？）が恣意的。δ_IPR の基底依存性と同型の「定義の任意性」問題が核側にも残っている。
2. **周期境界の問題**: 周期セルでは位置演算子（とその分散）が ill-defined。Resta が電子側で解決した問題（Resta 1998, Resta & Sorella 1999）の核版が未解決のまま。

## 2. 提案: z_nuc

[Resta & Sorella (1999)](https://arxiv.org/abs/cond-mat/9808151)（以下 RS）の電子局在位相（RS Eq. 10）

```
z_N = ⟨Ψ| exp(i(2π/L) X̂) |Ψ⟩,   X̂ = Σᵢ xᵢ
```

の核版を定義する:

```
z_nuc = ⟨ exp(i(2π/L) Σ_I X_I) ⟩       （多体版 — RS Eq. 10 の類推）
z₁    = ⟨ exp(i(2π/L) X_I) ⟩_{I,t}     （単粒子版 — RS Eq. 1 の類推: 分布 P(x) の特性関数）
```

- X_I: 核 I の位置（周期セル内）、⟨⟩: AIMD トラジェクトリ上の熱平均
- 単粒子版は z₁ = ∫dx e^{i(2π/L)x} P(x)（P: 核位置の一粒子確率分布）— RS Eq. (1) と同型

**原典で確定した規約**（RS 精読済み）:

```
単粒子・1D:  λ² = ⟨x²⟩ − ⟨x⟩² = −(L/2π)² ln|z|²          (RS Eq. 8)
多体・1D:    D = −lim_{N→∞} N ln|z_N|²,  λ = √D/(2πn₀)     (RS Eq. 11)
d 次元一般化: N を N^{2/d−1} に置換、λ²_xx = 4^{1/d−1} D_xx/(2πn₀^{1/d})²  (RS 一般化ノート (ii))
```

D は「局在なら有限、非局在なら発散」の判定量。**核に移植すると: 固体（核局在）で D_nuc 有限、液体で発散** — 「有限 vs 発散」の鋭い判定になる。なお分布の中心は z の位相から ⟨x⟩ = (L/2π) Im ln z（RS Eq. 6、Selloni et al. 1987 由来）。

## 3. 予想される振る舞い

| 相 | \|z_nuc\| | 物理 |
|----|-----------|------|
| 結晶 | 有限（> 0） | 原子が格子点近傍に局在。本質的に Debye-Waller 因子 |
| 液体 | → 0 | 分布の一様化。Bragg ピーク消失の定量化 |
| 気体 | → 0 | 同上 |

### 重要な制約（正直に書く）

**z_nuc は液体と気体を区別しない。** これは「固体か否か」の静的指標であり、δ_nuc の完全な置き換えではない。液/気の区別には依然として D（拡散係数）または δ_elec 側の情報が必要。したがって相図の軸構成は:

| 軸 | 量 | 区別するもの | 性格 |
|----|-----|------------|------|
| 1 | z_nuc | 固体 / 非固体 | 静的・厳密・物質固有 |
| 2 | ⟨r²⟩_c or Drude weight（2枚看板） | 絶縁体 / 金属 | 静的・厳密・物質固有（memo_kohn_resta_swm_lineage.md §3） |
| 3 | D | 液体 / 気体（連続） | 動的・補助 |

positioning_map.md §3.2 の3要件のうち、z_nuc は (1) 一粒子確率分布から計算可能・(3) 物質固有を満たすが、**(2) 固液気の連続パラメトリゼーションは単独では満たさない**。「単一変数で3相」から「厳密な2軸＋補助軸」へ設計変更する方が防御可能（memo_liquid_definition_gap_defense.md §2 の気液連続性とも整合 — 液/気を鋭く分かつ軸はそもそも存在しえない）。

## 4. 実験との接続（最大の魅力）

⟨exp(iQ·u)⟩ は結晶学の **Debye-Waller 因子そのもの**。つまり z_nuc は「Bragg ピークが生き残るか消えるか」の定量化であり、X線・中性子回折という既存実験と直結する。

- 固体: Bragg ピーク強度 ∝ |⟨exp(iG·u)⟩|² > 0
- 液体: Bragg 消失 = |z_nuc| → 0

「道具は全部あった。組み合わせ方が新しい」（positioning_map.md §3.3）のもう一つの実例: **Debye-Waller 因子（1913年から既知）を相の定義変数として読み替える。**

## 6. 新規性の状況（2026-07-04 に一次照合済み、論文化前に網羅的照合が必要）

- Debye-Waller 因子: 既知（1913〜）
- Resta 位相の電子への適用: 確立（1999〜）
- **核系に Resta 位相を適用して液体状態を定義した研究: 把握している限り存在しない**
- **[Grasselli & Baroni (2019) Nature Physics](https://doi.org/10.1038/s41567-019-0562-0)（[arXiv:1902.07256](https://arxiv.org/abs/1902.07256)）— 要旨照合済み**: 液体絶縁体の電荷輸送のトポロジカル量子化・ゲージ不変性（イオンの整数酸化数の理論）。Berry 位相を液体イオン系に使った最近傍の仕事だが、対象は**輸送係数**であり、z_nuc による**相分類**ではない → 新規性は保持。ただし引用必須（「Berry 位相的量が液体イオン系で well-defined」の先行として味方になる）。
- **Selloni, Carnevali, Car & Parrinello (1987) PRL 59, 823 — RS の ref [6] で発見**: z の**位相**（⟨x⟩ = (L/2π) Im ln z）を溶融塩中の単一量子粒子（電子）の追跡に使用。位相量を液体・無秩序系に適用した直接の先行。我々の新規性は「位相（中心）ではなく**モジュラス（広がり）を、電子ではなく核に、輸送ではなく相分類に**使う」と精密化される。
- **Sun (2024) [arXiv:2403.16868](https://arxiv.org/abs/2403.16868) — 要旨照合済み（2026-07-04）**: complex polarization（Resta 位相）の**有限温度**拡張。ただし電子のみ・モデル系・液体/融解/相分類はスコープ外。「z の有限温度化」の技術的先行として引用必須の味方。我々の一歩（核＋熱アンサンブル＋相分類）は空白のまま。
- **要照合（未確認）**: Onishi & Fu の "quantum weight"（PRX 2024 前後？）— ∫σ/ω 型の量を物質の量子性指標として提案しているはず。「sum rule 量を記述子として順方向利用する」精神が最も近い可能性。固体・T=0 の範囲か、液体に触れているかを確認すること。
- "configurational polarization" 系のキーワードでの網羅的検索は未了。

## 7. リスク

1. **多体版の有限サイズ減衰 → 解決済み（RS 精読、2026-07-04）**: RS Eq. (11) の規格化 D = −N ln|z_N|²（d 次元では N^{2/d−1}）がまさにこの問題への処方。局在状態では N ln|z_N|² が有限に収束（|z_N| → 1）、非局在で発散。核への移植はこの規格化をそのまま使えばよい。RS Fig. 2 に有限 N 収束の実例あり（N=8 で真値と 27% ずれる例も — 有限サイズ効果の目安として有用）。
2. **参照格子の必要性**: u_I は参照格子点からの変位として定義される。液体では参照格子が消える — G を「固体相の最小逆格子ベクトル」に固定するか、S(Q) の第一ピーク位置を使うか。設計の要。
3. **ガラス・過冷却液体**: |z_nuc| は有限（構造は凍結）→ ガラスは固体側に分類される。「ガラスは動力学的には固体」は defensible（memo_liquid_definition_gap_defense.md §6）だが明記が必要。
4. **液晶・塑性結晶**: 並進と配向の秩序が分離する中間相では軸が縮退する。スコープ外と明示。
