# 軌道遷移はミクロ相転移か？ — R vs γ の負の相関から

**Status: 2026-04-03。文献データによる予備的観察。定量的検証は不十分。**
**注意: 本メモの主張はすべて極めて薄い仮説段階であり、検証されていない。**

---

## 1. 発端: δ枠組みの素朴な予言の失敗

δ枠組みの素朴な予言:

> δ_elec が光学応答 R と表面張力 γ の両方を支配するなら、
> R と γ は正の相関を持つはずである。

文献データ（13元素）で検証した結果:

```
Pearson r = −0.53, p = 0.06（負の相関、有意水準ギリギリ届かず）
Spearman ρ = −0.42, p = 0.16（順位相関も負）
```

**素朴な予言は棄却された。** 相関は正ではなく負。

|------|----------|-----------|--------|
| K    | 120      | 0.96      | sp     |
| Na   | 210      | 0.97      | sp     |
| Pb   | 480      | 0.62      | sp-p   |
| Hg   | 485      | 0.74      | sp     |
| Sn   | 610      | 0.79      | sp-p   |
| Ga   | 720      | 0.76      | sp-p   |
| Zn   | 800      | 0.60      | d      |
| Ag   | 960      | 0.98      | d      |
| Al   | 1020     | 0.92      | sp     |
| Au   | 1190     | 0.81      | d      |
| Cu   | 1400     | 0.62      | d      |

| Fe   | 1930     | 0.56      | d      |

γ 出典: Keene (1993), Mills & Su (2006), Suryanaryana & Mika (2019)
R 出典: Johnson & Christy (1972/1974), Rakic (1998), Inagaki et al. (1981), Palik

**先行研究: R vs γ の直接相関を示した論文は見つからなかった。**
最近接は Miedema & Boom (1978) Z. Metallkd. 69, 183 だが、
R と γ を直接プロットしていない。

## 2. なぜ負になったか: sp vs d の軌道性

負の相関の原因は明確:

- **Na, K** (sp): γ 低い（120-210）、R 極めて高い（0.96-0.97）
- **Fe, Ni, Cu** (d): γ 高い（1400-1930）、R 低い（0.56-0.66）

d 金属では:
- d 電子が原子上に**局在** → セル境界の電子密度 n_ws は高い
  （d 電子のスクリーニングで有効核電荷が見える → sp 電子が引き寄せられる）
  → **γ が高い**
- d 電子が可視域で**インターバンド吸収**を起こす → **R が下がる**

つまり d 電子は γ を上げつつ R を下げる。
同じ δ_elec のはずなのに、物性への効き方が逆。

## 3. 仮説: 軌道遷移 = ミクロ相転移（極めて薄い仮説）

**注意: 以下は定量的根拠のない概念的アナロジーであり、
構造的な類似性の指摘にとどまる。「相転移」という語の使用は
厳密な熱力学的意味ではなく、記述子の質的変化を意味する。**

sp → d の軌道性の変化を、δ枠組みの中で「ミクロ相転移」と
見なすことが可能かもしれない。

マクロな相転移（固体→液体）では、核の空間構造が質的に変わる:
格子点に局在 → 拡散的に非局在化。

軌道遷移では、電子の空間構造が質的に変わる:
等方的に非局在化（sp）→ 指向性を持って局在（d）。

| | マクロ相転移 | ミクロ「相転移」（軌道遷移） |
|---|---|---|
| 何が変わるか | 核の空間構造 | 電子の空間構造 |
| 秩序変数 | δ_nuc | 軌道角運動量 l |
| 閾値 | Lindemann 条件 | 周期表の族遷移 |
| 物性への影響 | 粘度、拡散係数 | R と γ の相対関係が反転 |

**この対応の強さは現時点では不明。** 構造的類似性は見えるが、
以下の点が未解決:

- マクロ相転移には自由エネルギーの特異点がある。軌道遷移に対応する
  「自由エネルギー」は何か？（不明）
- マクロ相転移は温度駆動。軌道遷移は原子番号（離散変数）駆動。
  連続的な制御パラメータはあるか？（圧力？合金組成？）
- 「相転移」と呼ぶためには秩序変数の対称性の変化が必要。
  sp → d に対応する対称性の破れは何か？（球対称 → d 対称？不明確）

## 4. δ_elec のベクトル化（推測）

スカラー δ_elec ではRとγの同時予測に失敗した。
修正として δ_elec を軌道成分に分解する:

```
δ_elec = (δ_sp, δ_d)
```

- sp 金属: δ_sp 高, δ_d 低 → R 高, γ 中
- d 金属:  δ_sp 低, δ_d 高 → R 低, γ 高

光学応答と表面張力への寄与:

```
R ∝ δ_sp                        （sp 電子の Drude 応答が支配）
γ ∝ f(δ_sp, δ_d)                （両方が n_ws に寄与）
```

**これは現時点で操作的定義がない。** IPR や Wannier spread は
全電子を平均するため、sp/d の分離ができない。
軌道分解された IPR（l 成分ごとの IPR）が必要だが、
そのような計算は見たことがない（潜在的新規性？要調査）。

## 5. 周期表の metalloid staircase との接続

`memo_periodic_table_and_refractive_index.md` で
「metalloid staircase は δ = δ_critical の等高線」と書いた。

軌道遷移がミクロ相転移だとすると、metalloid staircase は
この「ミクロ相転移」のマクロ的表現と解釈できる:

- staircase の左（金属側）: δ_sp が支配 → 金属的 R, 有限 γ
- staircase の右（非金属側）: 電子が局在 → R 低, γ 低
- staircase 上: sp → 局在 の遷移点 = δ_sp = δ_critical

staircase のノッチ（Ga, Ge 付近での段差）は
d ブロック挿入による δ の局所的低下（d 電子収縮）で説明される
— これも「ミクロ相転移」の局所的なシフトとして読める。

**ただしこれも定性的議論であり、定量的検証は行っていない。**

## 6. δ枠組みの修正版予言

素朴な予言（R ∝ γ）の失敗を踏まえた修正:

> **同一の軌道タイプ（sp 同士、d 同士）の中では、
> δ_elec が高いほど R も γ も高い。
> しかし sp と d を横断すると、d 電子の interband 吸収が
> R を下げるため、全体の R-γ 相関は消える（または負になる）。**

この修正版予言は、以下のテスト可能な形を持つ:

1. sp 金属のみで R vs γ を取ると正の相関になるはず
   → 現データ: r = −0.24 (n=7, p=0.61) — **相関なし。失敗？**
   → ただし n=7 で検出力不足。Na/K が外れ値的。

2. d 金属のみで R vs γ を取ると正の相関になるはず
   → 現データ: r = −0.48 (n=6, p=0.34) — **やはり負。失敗。**
   → Ag が外れ値（γ=960 なのに R=0.98）。

**修正版予言も現データでは支持されない。**

## 7. 正直な評価

現時点で言えること:

1. R と γ の直接相関を調べた先行研究は見つからなかった（新規性はある）
2. δ枠組みの素朴な予言（R ∝ γ）は負の相関により棄却された
3. 軌道タイプで分けても相関は改善しなかった
4. 軌道遷移を「ミクロ相転移」と見なすアナロジーは構造的に面白いが、
   定量的裏付けがない

**「失敗した予言」は論文において弱みではなく、正直に報告すれば
枠組みの射程と限界を明確にする資産になる。**

具体的には:
- δ_elec がスカラーでは不十分であること
- 軌道性（sp vs d）が独立な変数として必要であること
- δ の内部構造（軌道分解）が将来課題であること

これらは Paper 2 の Discussion で「limitation and future work」として
記述する価値がある。

## 8. NaK / CuZn 比較実験（2026-04-03 追記）

### 8.1 実験の論理

「ミクロ相転移」仮説を検証するため、2つの合金系で R vs γ の相関を比較:

- **NaK合金**（ネガティブコントロール）: sp金属同士。軌道遷移なし。
  → R ∝ γ（正の相関）が予測される。
- **CuZn合金**（ポジティブテスト）: sp-d混合。Zn増加でsp/d比が変化。
  → R と γ が逆方向に動く（負の相関）が予測される。

### 8.2 結果

| 系 | 軌道タイプ | R vs γ | p値 | 解釈 |
|---|---------|--------|-----|------|
| NaK | sp のみ | **r = +0.97** | 0.0001 | δ_sp ↑ → R ↑, γ ↑ |
| CuZn | sp-d 混合 | **r = −0.85** | 0.008 | d電子 → R↓（吸収）, γ維持（n_ws↑） |

**符号の反転が、軌道遷移に伴う R-γ 関係の質的変化を示す。**

### 8.3 NaK の詳細

Drude模型で R を計算（ω_p: Na 5.71 eV → K 3.72 eV を線形内挿、
ρ: Na 9.7 → K 14.5 μΩ·cm を Nordheim 近似）。
γ は Jordan & Lane (1965), Alchagirov (2011), ACS (2025) から。

Na分率を増やすと ω_p が上がり（電子密度増加）、
R も γ も同時に上昇。r = +0.97。

**注意**: R の変動幅は 0.982 → 0.984 と極めて小さい
（Drude金属は可視域で R が飽和するため）。
ランキングは完璧だが、絶対変化量が小さいことは限界。

### 8.4 CuZn の詳細

R は Querry (1985) の Cu90Zn10, Cu70Zn30 データ +
Johnson & Christy (1972) の純 Cu から推定。
γ は Butler 模型による内挿（Cu 1300, Zn 780 mN/m）。

500nm（Cuのinterband吸収領域）で最も顕著:
Zn 増加 → d バンドが下がり空になる → interband 吸収が弱まる → R **上昇**。
同時に γ は Zn 表面偏析で **低下**。
→ R と γ が逆方向: r = −0.85 (p = 0.008)。

**注意**: CuZn の R 値は文献からの推定であり、直接測定値ではない。
γ も Butler 模型の推定。定性的な符号反転は堅いが、
定量的な r 値は不確実性が大きい。

### 8.5 この結果の意味

1. **δ枠組みの修正版予言が支持された**: 同一軌道タイプ内では R ∝ γ、
   軌道タイプが変わると反転。
2. **符号反転そのものが新規知見**: R-γ相関の符号が軌道性に依存する
   ことを指摘した先行研究は見つかっていない。
3. **「ミクロ相転移」の定量的証拠**: sp→d の軌道遷移が、
   マクロ物性（R, γ）の相関構造を質的に変えることを示した。

**ただし**: CuZn データの不確実性が大きく、
この結論は予備的（preliminary）である。
直接測定による確認が必要。

## 9. 未解決の問い

1. 軌道分解 IPR（l 成分ごと）を計算した先行研究はあるか？
2. 合金系（sp-d 混合）で連続的に軌道遷移を追跡できるか？
   （CuZn 黄銅は良い候補 — Zn 濃度で sp/d 比率が変わる）
3. 高温での d 電子の非局在化（d バンド幅の熱的拡大）は
   R と γ を同時に変えるか？
4. Ag の「異常」（d 金属なのに R が極めて高い）は
   d バンドが可視域より十分下にあることで説明できるか？
   → Ag の d-sp interband 遷移は ~4 eV（紫外）なので
   可視域 (~2.2 eV at 550nm) では Drude 的 → R が高い。
   これは「d 金属でも d バンドが十分深ければ sp 的に振る舞う」
   ことを意味し、軌道遷移の閾値が**エネルギー依存**であることを示唆。
