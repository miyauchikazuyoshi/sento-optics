## 1. 屈折率の予測には δ 単独では不十分

### 実験結果（Group 14 同構造比較）

| 元素 | 周期 | Ω/WF (Å²) | E_g (eV) | n (文献) | n²−1 |
|------|------|-----------|----------|---------|------|
| C (diamond) | 2 | 0.954 | 5.5 | 2.42 | 4.86 |
| Si | 3 | 3.049 | 1.1 | 3.44 | 10.83 |
| Ge | 4 | 2.275 | 0.67 | 4.0 | 15.0 |

Ω/WF の序列: C < Ge < Si
n の序列: C < Si < Ge

**Ge の Ω/WF が Si より小さいのに n は大きい。**
δ_elec 単独では屈折率の元素間比較を再現できない。

### 同一元素内（炭素同素体）では δ の序列が一致する理由

同一元素なら原子分極率 α とバンドギャップ Eg の寄与が
構造の違いだけに帰着するため、δ の序列 ≈ 光学応答の序列。
Paper I の 6/7 分類成功はこの条件下での結果。

### 異元素間で必要な補正：三因子モデル

Penn model を基にすると：

```
n² − 1  ∝  δ_elec × α_atom / E_g²
```

| 因子 | 物理 | C vs Si vs Ge |
|------|------|---------------|
| δ_elec (Ω/WF) | 電子の非局在化度 | C < Ge < Si |
| α_atom | 原子分極率（原子サイズ依存） | C < Si < Ge |
| 1/E_g² | 光学遷移の容易さ | C << Si < Ge |

Ge は δ が Si より小さいが、α が大きく E_g が小さいため、
積 δ × α / E_g² は Si を上回る → n(Ge) > n(Si)。

### 「単体の δ」と「群としての δ_eff」

- **単体（孤立原子・同一元素）**: δ_elec だけで序列が決まる
- **群（異元素の固体）**: δ_eff = δ_elec × α / E_g² のような
  複合量が必要

シミュレーションでは：
1. Ω/WF は現パイプラインのまま（Wannier90）
2. E_g は DFT バンド構造から（すでに計算内に含まれる）
3. α は文献値 or DFT 誘電関数 (epsilon.x) から取得
4. 三因子の積で n を予測し、文献値と照合

## 2. d電子遮蔽効果の確認

### Paper 2 の知見の再現

Paper 2 で示したこと：
- sp 金属（Na, K, Al）: 価電子が非局在化 → 高い n_ws
- d 金属（Cu, Zn）: d 電子の遮蔽で sp 軌道が局在化

今回の Wannier 計算で同じ効果を確認：
- Si (3周期, d なし): Ω/WF = 3.05 → sp3 が広がれる
- Ge (4周期, 3d¹⁰ あり): Ω/WF = 2.28 → 3d 遮蔽で sp3 が収縮

**同じ結晶構造・同じ族なのに、d 遮蔽で Ω が 25% 減少。**
Paper 2 の価電子分解の知見が、光学系でも整合的に再現された。

### Ge のバンド構造で確認した d バンドの位置

```
バンド 1-10:  -16.33 ~ -16.07 eV  ← 3d バンド（10本、非常に狭い＝局在）
バンド 11:    -4.29 eV             ← sp3 価電子帯下端
バンド 12-14: ~8.5 eV              ← sp3 価電子帯上端
```

3d バンドは -16 eV 付近に 10 本が 0.3 eV 幅に密集。
完全に局在した芯電子的振る舞い。しかしこの局在した d 殻が
外側の sp3 軌道を遮蔽・収縮させている。

## 3. 周期表の金属-非金属境界（半金属階段）

### 半金属の階段 (metalloid staircase)

```
     B
      Si
       Ge As
          Te
           At
```

金属-非金属の境界が周期表の左上から右下に走る。

### δ による説明

- **金属**: δ_elec 高い（価電子が非局在化）
- **非金属**: δ_elec 低い（価電子が局在化）
- **半金属**: δ_elec ≈ δ_critical（境界上）

**階段の形は δ_elec = δ_critical の等高線。**

### 等高線の形を決める 3 つの力

| 効果 | 周期表上の方向 | δ への影響 |
|------|---------------|----------|
| 殻数の増加 | ↓（下に行く） | δ↑（原子大、非局在化しやすい） |
| 有効核電荷 Z_eff 増加 | →（右に行く） | δ↓（核が電子を引きつけ局在化） |
| d 遮蔽 | 4 周期以降 | δ↓（d 殻が sp 軌道を収縮） |

### なぜ階段は右下に降りるか

下に行くと殻が増えて δ が上がる。
同じ δ_critical に達するには、もっと右（もっと高い Z_eff）まで
行かないといけない。→ 境界が右に移動。

### なぜ階段は直線じゃないか

d ブロック挿入（4 周期）で Z_eff が不連続に増加する（d ブロック収縮）。
Ge, As は 3d¹⁰ の遮蔽を受けて「余計に局在化」→ 階段に段差が生じる。

### 今回のデータによる実証

```
Si (3周期, d なし):  Ω/WF = 3.05  ← 非局在化度が高い → 半導体
Ge (4周期, 3d¹⁰):   Ω/WF = 2.28  ← d 遮蔽で局在化 → ギリギリ半導体
```

同じ第14族で同じ結晶構造なのに、Ge は Si より 25% 局在化。
これが Ge が Si より「非金属寄り」である電子構造的理由。

### 第14族を下に辿ると

| 元素 | 周期 | d 遮蔽 | 予測 | 実際 |
|------|------|--------|------|------|
| C | 2 | なし | δ 最小 → 絶縁体 | ✅ ダイヤモンド |
| Si | 3 | なし | δ 中 → 半導体 | ✅ 半導体 |
| Ge | 4 | 3d¹⁰ | δ 凹み → 半導体（ギリギリ） | ✅ 半導体 |
| Sn | 5 | 4d¹⁰ | δ 再上昇 → 半金属/金属 | ✅ α-Sn 半金属, β-Sn 金属 |
| Pb | 6 | 5d¹⁰+4f¹⁴ | δ 十分高い → 金属 | ✅ 金属 |

**δ の非単調変化（d 遮蔽による凹み）が半金属階段の「段差」を生む。**
