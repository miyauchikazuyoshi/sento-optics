## 1. 核心: coth 公式による δ_nuc の統一的定義

調和振動子の有限温度での変位の二乗平均:

```
⟨u²⟩(T) = (ℏ / 2Mω_D) · coth(ℏω_D / 2k_BT)
         = (ℏ / 2Mω_D) · coth(Θ_D / 2T)
```

これを格子間距離 a で無次元化して δ_nuc を定義:

```
δ_nuc(T) = ⟨u²⟩(T) / a² = δ_ZP · coth(Θ_D / 2T)

ここで δ_ZP = ℏ / (2Mω_D a²)  [零点振動の寄与]
```

**融解条件（一般化 Lindemann）:**

```
δ_nuc(T_m) = δ_ZP · coth(Θ_D / 2T_m) = δ_c  ≈ 0.01-0.02
```

### 極限

- **古典的融解** (Θ_D << 2T_m): coth → 2T_m/Θ_D → δ_nuc ∝ T
  - 通常の金属・半導体の融解。温度が上がると振幅が増え、臨界値に達して融ける。
- **量子融解** (Θ_D >> 2T_m): coth → 1 → δ_nuc = δ_ZP
  - 零点振動だけで臨界値に達する。温度によらず融ける（He の加圧融解）。

**この一つの式が量子融解と古典融解を連続的に接続する。**

## 3. de Boer パラメータ Λ*

もう一つの量子性指標:

```
Λ* = h / (σ √(mε))
```

- σ: LJ サイズパラメータ（原子直径）
- ε: LJ 井戸深さ（相互作用強度）

| 物質 | Λ* | 量子性 |
|------|-----|-------|
| He-3 | 3.1 | 最強 |
| He-4 | 2.6 | 極めて強い |
| H₂ | 1.7 | 強い |
| Ne | 0.59 | 中程度 |
| Ar | 0.19 | 弱い |
| Kr | 0.10 | 無視可能 |

物理的意味: de Broglie 熱波長と原子間距離の比。
Λ* >> 1 で波束が重なり合い量子効果支配。

## 4. Θ_D — E_g — Ω の連鎖

### Θ_D と E_g の経験的関係

Ullrich, Singh & Vopson, AIP Advances 7, 045109 (2017):

```
Θ_D = T_0 · exp(γ · E_g)

III-V化合物: T_0 = 272 K, γ = 0.018 /meV
II-VI化合物: T_0 = 169 K, γ = 0.015 /meV
```

| 物質 | E_g (eV) | Θ_D (K) |
|------|----------|---------|
| Sn (grey) | 0.08 | 200 |
| Ge | 0.66 | 374 |
| Si | 1.12 | 645 |
| C (diamond) | 5.47 | 2230 |

**物理的根拠**: 強い結合 → 硬いフォノン（高 Θ_D）かつ大きいバンドギャップ（大 E_g）

### E_g と Ω の関係

Cardenas-Castillo et al. (2024): Ω_I = ∫ε₂/ω dω

E_g が大きい → ε₂ の吸収開始が高エネルギー → 一般に Ω_I が小さい

### 間接的連鎖

```
Θ_D ↑ ←→ E_g ↑ ←→ Ω ↓

すべて「結合の硬さ」を共有する。
```

**重要な帰結**: δ_elec (Ω) と δ_nuc (⟨u²⟩, Θ_D 経由) は独立ではない。
(δ_elec, δ_nuc) の相図上で、物質が占められる領域は制約される。
強結合材料（ダイヤモンド）は δ_elec も δ_nuc も小さく、
弱結合材料（He, 液体金属）は両方が大きい傾向がある。

## 7. Paper III 中心仮説: δ による量子-古典横断的物性分類

### 仮説の核

δ（非局在度）は確率分布の第二モーメント（分散）として定義され、
量子系でも古典系でも同一の数学的構造を持つ:

```
電子: δ_elec = Ω = ⟨r²⟩ - ⟨r⟩²     （|ψ|² の分散 — 量子的）
核:   δ_nuc = ⟨u²⟩/a²               （ボルツマン分布の分散 — 古典的にも量子的にも）
```

δ は「粒子がどこにいるかの不確定さ」を測る。
その起源が量子（ℏ）か古典（kT）かは問わない。

### 古典系で「量子的に見える」現象の説明

δ_nuc が格子間距離 d に近づくと:

```
δ_nuc << d: 粒子は離散的な散乱体 → 個別粒子の描像が有効
δ_nuc ~ d:  粒子の「居場所」が隣と区別不能 → 連続体近似が成立
              → コヒーレントな集団応答が可能に
```

この「離散→連続」転移は、量子系でのバンド形成（波動関数の重なり）と
**構造的に同型**。量子系では ℏ が広がりの下限、古典系では kT が広がりを駆動する。
coth 公式がその両者を一本で繋ぐ:

```
δ_nuc(T) = δ_ZP · coth(Θ_D / 2T)
              ↑         ↑
           量子的下限   古典的増幅
```

### 具体的な予測

1. **液体の表面**: δ_nuc ≈ d で分子位置が不確定
   → 界面が連続体として振る舞う → フレネル反射が成立
   （古典系なのにコヒーレント反射）

2. **液体金属**: δ_elec (Ω) も δ_nuc も両方大きい
   → 電子も核も非局在 → 高い表面張力 + 金属光沢が液体でも維持

3. **超流動 He**: δ_ZP だけで δ_nuc > d
   → 古典的な「粒子」の描像が完全に崩壊
   → 量子コヒーレンスが巨視的に出現

### 重要な注意: 古典的非局在 ≠ 量子コヒーレンス

古典液体の δ_nuc ~ d は、統計的平均による連続体化であって、
波動関数の位相コヒーレンスではない。
両者の区別は coth 公式で定量化できる:

```
Θ_D / 2T >> 1: coth → 1 → δ_nuc ≈ δ_ZP → 真の量子コヒーレンス（He）
Θ_D / 2T << 1: coth → 2T/Θ_D → δ_nuc ∝ T → 古典的統計平均（通常の液体）
```

「量子的に見える」古典現象は、量子コヒーレンスの模倣（mimicry）であって同一ではない。
しかし δ という共通言語で記述できることに変わりはない。

### δ とエントロピーの接続

δ が確率分布の分散であるなら、エントロピーとの関係は準恒等式的:

```
電子: S_elec/k_B = (1/2)ln(2πe·Ω)    （ガウス近似）
核:   S_nuc/k_B → ln(kT/ℏω) ∝ ln(δ_nuc)  （高温極限）

共通構造: S/k_B ∝ ln(δ)
```

もしこの関係が定量的に成り立つなら:
- 表面張力: γ ∝ ΔS/ΔA ∝ Δln(δ)/ΔA
- 融解条件: ln(δ_liquid) - ln(δ_solid) = ΔH/(k_BT_m)
- 光学応答: exp(2S_elec/k_B) ∝ Ω ∝ ∫ε₂/ω dω

光学も表面張力も融解も、δ を通じたエントロピーの異なる表現として統一される。

**注意**: S ∝ ln(δ) はガウス近似（調和近似）のもとで成立。
強い非調和性や多体相関が重要な系では、分布の形状情報（歪度、尖度）が
効いてくるため、δ（分散）だけでは S を捉えきれない。
Paper III での検証項目: 実際の物質で S ∝ ln(δ) の精度を評価すること。

### δ の無次元性と確率幾何学的解釈

本議論に現れるすべての量は無次元比である:

```
δ_elec = Ω/a²         （広がり/格子定数²）
δ_nuc = ⟨u²⟩/a²       （変位²/格子定数²）
Lindemann = √⟨u²⟩/d   （変位/格子間距離）
coth(Θ_D/2T)          （温度比）
T_m/Θ_D               （温度比）
S/k_B                  （エントロピー/ボルツマン定数）
```

δ は特定の物理法則（Schrödinger 方程式、分配関数）の詳細に依存せず、
**確率分布の形状に関する幾何学的な量**として定義されている。

「粒子の広がり / 系の特徴的長さ」が閾値を超えるかどうか —
この無次元比だけで融解・光学応答・表面張力を分類できるなら、
δ は量子力学や統計力学の**上位にある確率論的構造**の不変量を
物性の言葉で読んだものと解釈できる。

情報幾何学（Fisher 計量、統計多様体）との対応:
- δ（分散）は Fisher 計量の最も単純な成分
- S ∝ ln(δ) は統計多様体上の「体積」に対応
- 相転移は統計多様体上の特異点として現れる

### Paper III の主張（案・改訂）

> δ（確率分布の第二モーメント）は、量子系でも古典系でも
> 「粒子の空間的不確定さ」を測る無次元量であり、
> 系の特徴的長さとの比が物性を決定する。
>
> 電子の δ_elec (Ω/a²) は光学応答と表面張力を、
> 核の δ_nuc (⟨u²⟩/a²) は融解条件と相の分類を支配し、
> S ∝ ln(δ) を通じて両者は熱力学的エントロピーに接続される。
>
> coth 公式が量子極限（ℏ）と古典極限（kT）を連続的に繋ぎ、
> δ は量子力学・統計力学・熱力学に共通する確率論的構造の
> 最も基本的な不変量を物性の言語で表現したものである。

### なぜ第2モーメントが物理の基本量たり得るか（2026-04-03 追記）

上の主張には「なぜ第2モーメントか？ 第1モーメントではなく？」
という問いが潜んでいる。その回答:

**量子力学は、第1モーメントが原理的に特定できないから、
第2モーメントを計算軸にした。**

不確定性原理 ΔxΔp ≥ ℏ/2 は、⟨x⟩ を任意精度で知ることを禁じる。
物理学はこの制約のもとで、⟨x²⟩-⟨x⟩²（= 分散 = 広がり）を
基本的な記述量として採用し、これで量子系の物性を予測することに成功した:

- 電子の存在確率の広がり → 化学結合、バンド構造
- 波動関数の空間的分散 → トンネル効果、伝導
- Wannier spread Ω = ⟨r²⟩-⟨r⟩² → 分極、誘電応答

**δ 枠組みの核心主張: この戦略は古典系にも有効である。**

古典系では ⟨x⟩ は原理的に特定できる。しかし界面・相転移・輸送など
「広がりが勝負を決める現象」では、第1モーメント（粒子がどこにあるか）
よりも第2モーメント（粒子がどれだけ広がっているか）の方が
物性の予測力が高い:

```
量子:  ⟨x⟩ 不定 → ⟨x²⟩-⟨x⟩² で記述するしかない → 成功した
古典:  ⟨x⟩ 特定可能 → でも ⟨x²⟩-⟨x⟩² で記述した方が強い場面がある
```

coth 公式はこの二つの理由を一本の数式で繋ぐ:

```
δ_nuc(T) = δ_ZP · coth(Θ_D / 2T)

T → 0:  δ → δ_ZP   （量子: 第1モーメント不定、第2モーメントが残る）
T → ∞:  δ → 2T/Θ_D  （古典: 第1モーメントは定まるが、第2モーメントが予測力を持つ）
```

低温極限と高温極限で「第2モーメントを使う理由」は異なる。
しかし数式は連続的に一本。これは偶然ではなく、
**第2モーメントが両方の領域で物性の本質的情報を担っている**からである。

δ の汎用性が確率分布そのものと同程度に高い理由も、ここから理解できる。
δ は確率分布 p(x) から取れる**最小限の非自明な情報**
（第0モーメント=規格化、第1モーメント=平均位置 を引いた後に
残る最初の構造量）であり、確率分布が定義できる場所すべてに
δ が定義できる。物理ではあらゆる系が確率分布を持つ。

## 9. 拡散方程式による δ_nuc の定式化（未解決問題1への回答案）

**Status: 2026-04-03 の議論メモ。仮説段階。**

### 9.1 核心: 拡散方程式が δ_nuc の支配方程式

coth 公式（Section 1）は **固体中の振動**（⟨u²⟩が有界）を記述する。
液体では核が格子点を離れて**拡散**する。この遷移を統一的に記述するには、
拡散方程式を δ_nuc の支配方程式として位置づける必要がある。

拡散方程式:
```
∂n/∂t = D∇²n
```

自由拡散の解（グリーン関数）はガウシアンで、空間分散が時間成長する:
```
⟨r²(t)⟩ = 2dDt    （d次元）
```

これは Wannier spread Ω = ⟨r²⟩ - ⟨r⟩² と**同じ数学構造**（確率分布の第2モーメント）。
つまり δ_elec も δ_nuc も「確率分布の空間的広がり」として統一される。

| | 電子 (δ_elec) | 核 (δ_nuc) |
|---|---|---|
| 支配方程式 | Schrödinger 方程式 | 拡散方程式 |
| 非局在化の尺度 | Wannier spread Ω | MSD ⟨r²(t)⟩ |
| 固体での状態 | Ω 有界（局在 WF） | ⟨u²⟩ 有界（格子振動 → coth 公式） |
| 液体での状態 | Ω 増大（広がった WF） | ⟨r²(t)⟩ ∝ t（拡散、発散） |

### 9.2 格子拡散と電子タイトバインディングの同型性

拡散方程式を格子上で離散化すると:
```
∂n_i/∂t = Γ Σ_{j∈nn(i)} (n_j - n_i)
```

ここで Γ はジャンプ頻度、和は最近接格子点 j について取る。
連続極限では D = Γa²/2d（a: 格子定数、d: 次元）。

これは電子のタイトバインディング模型と**構造的に同型**:

| | 電子（TB） | 核（格子拡散） |
|---|---|---|
| 格子点の係数 | ホッピング積分 t | ジャンプ頻度 Γ |
| 局在の原因 | ポテンシャル障壁 / バンドギャップ E_g | 活性化障壁 E_a |
| 非局在化方向 | D_eff（有効伝導次元） | 配位数 z（拡散経路数） |
| 温度依存性 | Fermi-Dirac 分布 | Arrhenius: Γ = ν₀ exp(-E_a/kT) |

**タイトバインディングの t と格子拡散の Γ は同じ役割を果たす**:
どちらも「隣の格子点にどれだけ漏れ出すか」を決める係数。

### 9.3 E_a と E_g の対応

活性化障壁 E_a は δ_nuc における「バンドギャップ」に相当する:

- E_g 大 → 電子が動けない → δ_elec 低
- E_a 大 → 核が動けない → δ_nuc 低

さらに配位数 z は D_eff（有効伝導次元）に対応する:
配位数が高い格子ほど拡散経路が多い → 核の非局在化が促進される。

### 9.4 coth 公式と拡散方程式の接続

coth 公式（振動）と拡散方程式（拡散）は異なる時間スケールの現象だが、
格子拡散の枠組みで統一できる可能性がある:

- **短時間** (t << 1/Γ): 核は格子点周りで振動。⟨u²⟩ は coth 公式に従う。
- **長時間** (t >> 1/Γ): 核が隣接格子点にジャンプ。⟨r²⟩ ∝ t の拡散的成長。
- **融解点**: Γ が急増（E_a の実効的消失）し、振動レジームから拡散レジームへ転移。

つまり coth 公式は格子拡散の **短時間極限**（ジャンプが起きない時間スケール）
として自然に回収される。液体状態は **長時間極限**（多数ジャンプ後）。

### 9.5 δ_nuc の無次元化（未解決）

拡散係数 D から δ_nuc をどう構成するかは未確定。候補:

| 候補 | 定義 | 利点 | 問題 |
|------|------|------|------|
| D そのもの | 拡散係数 [m²/s] | 直接測定可能 | 次元がある |
| D/D₀ | D₀ = 自由粒子の拡散係数 | 無次元、0-1 にマップ可能 | D₀ の定義が曖昧 |
| ⟨r²(τ)⟩/a² | 特性時間 τ での MSD / 格子定数² | Lindemann と直結 | τ の選び方が恣意的 |
| coth 連結 | δ_ZP·coth(Θ_D/2T) (固体), 6Dt/a² (液体) | 両レジーム包含 | 遷移点での接続が不連続 |

### 9.6 Paper 3 への含意

この同型性が成立すれば、Paper 1-2 で確立した δ_elec の数学的装置
（IPR, Wannier spread, バンド幅）が δ_nuc にそのまま転用できる。

具体的には:
- 核の「タイトバインディング」= 格子拡散模型
- 核の「IPR」= 核密度分布の逆参加率
- 核の「Wannier spread」= MSD ⟨r²(t)⟩
- 核の「バンドギャップ」= 活性化障壁 E_a

**δ_elec と δ_nuc が数学的に同型であることを示せれば、
(δ_nuc, δ_elec) 相図の二軸が同じ言語で書かれていることの正当化になる。**
