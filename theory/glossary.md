# Glossary / 用語表

本研究で使用する主要な物理量・概念の定義と、論文内での位置づけを整理する。

---

## 分類変数（論文で実際に使用）

### E_g — バンドギャップ (Band gap)

- **定義**: 価電子帯上端と伝導帯下端のエネルギー差。E_g = min(E_c(k)) - max(E_v(k))
- **単位**: eV
- **物理的意味**: 電子が光を吸収するために必要な最小エネルギー。E_g > 3.1 eV なら可視光を吸収できず透明になる。
- **論文での役割**: Decision Tree の第一分岐変数。「どのエネルギーで光学応答が起きるか」を決定する。
- **限界**: E_g ≈ 0 の系（金属、グラファイト）を区別できない。液体系では定義困難。

### D_eff — 有効伝導次元 (Effective conduction dimensionality)

- **定義**: 群速度テンソル ⟨v_α v_β⟩ の有効ランク。
  - v_α(k) = ℏ⁻¹ ∂E(k)/∂k_α（群速度）
  - T_αβ = (1/N_k) Σ_k Σ_{n,m} v_nm^α · conj(v_nm^β)
  - D_eff = Tr(T)² / Tr(T²)
- **取りうる値**: 0（分子系）、1（1D鎖）、2（層状）、3（等方的3D）
- **物理的意味**: 電子がどの方向に自由に動けるかの次元数。結晶学的次元とは異なる（例: h-BN は2D結晶だが D_eff = 0）。
- **論文での役割**: Decision Tree の第二分岐変数。E_g ≈ 0 の系で「黒 vs 光沢」を区別する。
- **計算方法**: バンド構造 E(k) から速度行列要素を求め、テンソルのランクを計算。DFT/Wannier/タイトバインディングで実行可能。

---

## 概念的変数（動機づけとして使用、分類には未使用）

### δ — 非局在化指数 / 粒子の自由度 (Delocalization index / Particle freedom)

- **概念**: 構成粒子（電子、原子核）がどれだけ自由に動けるかの度合い。
- **操作的定義**: 一意ではない。以下のプロキシ指標の相互整合性から間接的に推定する。
- **論文での役割**: D_eff の導入を動機づけた物理的直感。「光学応答が粒子の自由度で決まるなら、自由度の方向性（D_eff）も重要」という着想の源泉。分類変数としては使用していない。
- **将来課題**: 液体系・界面系で独立に検証する必要がある。E_g が定義困難な系での分類に δ が不可欠になる可能性がある。

### δ_elec — 電子の非局在化度 (Electronic delocalization)

- **概念**: 電子の波動関数がどれだけ広がっているか。
- **固体での対応**: バンド幅 W、有効質量 m*、IPR などで定量化可能。
- **論文での位置づけ**: 固体系では E_g と強く相関するため、独立変数としての検証が困難。

### δ_nuc — 核の非局在化度 / 流動性 (Nuclear delocalization / Fluidity)

- **概念**: 原子核がどれだけ自由に動けるか。固体では凍結（δ_nuc ≈ 0）、液体では有限。
- **論文での位置づけ**: 本研究では検証していない。液体の光沢（水面、液体ガリウム）を説明する将来の拡張変数。

## δ のプロキシ指標

### W_π — π帯バンド幅 (π-band width)

- **定義**: π電子バンドのエネルギー幅 W = E_max - E_min
- **単位**: eV
- **値の例**: グラフェン 9.0 eV, C60 0.45 eV, h-BN 2.4 eV
- **δ との関係**: 大きい W_π → 大きい δ（電子が広く非局在化）

### 1/m* — 逆有効質量 (Inverse effective mass)

- **定義**: バンド端での曲率の逆数。1/m* = ℏ⁻² ∂²E/∂k²
- **単位**: m₀⁻¹（自由電子質量の逆数）
- **δ との関係**: 大きい 1/m* → 大きい δ（電子が軽く動きやすい）

### IPR — 逆参加率 (Inverse Participation Ratio)

- **定義**: IPR_i = Σ|ψ_i(r)|⁴ / (Σ|ψ_i(r)|²)²
- **範囲**: 1/N（完全非局在化）～ 1（完全局在化）
- **δ への変換**: δ_norm = 1/(N × ⟨IPR⟩)
- **計算方法**: タイトバインディングまたは DFT の固有ベクトルから直接計算

## 光学量

### ε(ω) — 誘電関数 (Dielectric function)

- **定義**: ε(ω) = ε₁(ω) + iε₂(ω)。物質の電磁場への応答を記述する複素関数。
- **ε₂(ω)**: 吸収に対応。ε₂ が可視域で大きい → 光を強く吸収。
- **ε₁(ω)**: 分散に対応。屈折率に寄与。
- **論文での位置づけ**: 固体系では ε(ω) が光学応答を完全に記述する。E_g + D_eff は ε(ω) の粗視化（coarse-graining）。液体系では ε(ω) の微視的起源が不明確であるため、E_g + D_eff が独自の分類手段になり得る。

### σ(ω) — 光学伝導度 (Optical conductivity)

- **定義**: クボー公式による。σ(ω) ∝ Σ_{n,m,k} |⟨nk|v|mk⟩|² δ(E_m - E_n - ℏω)
- **ε₂ との関係**: ε₂(ω) = σ(ω) / (ε₀ω) （SI単位系）
- **グラフェンの普遍値**: σ₀ = πe²/(2h) ≈ 6.08 × 10⁻⁵ S（周波数に依存しない）

### R(ω) — 反射率 (Reflectivity)

- **定義**: フレネル方程式から。R = |(√ε - 1)/(√ε + 1)|²（垂直入射）
- **範囲**: 0（完全透過）～ 1（完全反射）
- **光沢との関係**: R が可視域で高い → 鏡面反射 → 光沢

### R_vis — 可視域平均反射率 (Visible-range average reflectivity)

- **定義**: R_vis = ⟨R(ω)⟩ for 1.6 eV ≤ ω ≤ 3.1 eV（400-780 nm）
- **論文での使用**: δ × D_eff との相関を検証するための集約量

## 分類カテゴリ

### Decision Tree の出力

| カテゴリ | 条件 | 例 |
|---------|------|-----|
| 透明 (Transparent) | E_g > 3.1 eV | ダイヤモンド, h-BN |
| 色 (Colored) | 0 < E_g < 3.1 eV | C60, 半導体型SWCNT |
| 黒+光沢 (Black with luster) | E_g ≈ 0, D_eff = 1-2 | グラファイト, 金属型SWCNT |
| 金属光沢 (Metallic luster) | E_g ≈ 0, D_eff = 3 | 金, 銀, 銅 |

## モデル・手法

### タイトバインディング (Tight-binding, TB)

- **概念**: 原子軌道の線形結合でバンド構造を記述する近似法。
- **本研究での使用**:
  - 2バンドモデル（π帯のみ）: グラフェン, 1D鎖
  - 8バンド Slater-Koster モデル（s, p_x, p_y, p_z）: グラフェン, ダイヤモンド
- **限界**: パラメータ依存。バンドギャップを過小評価する傾向（LDA的）。

### Slater-Koster モデル

- **概念**: 2中心積分の方向依存性を方向余弦 (l, m, n) でパラメトライズした TB 法。
- **パラメータ**: V_ssσ, V_spσ, V_ppσ, V_ppπ + オンサイトエネルギー ε_s, ε_p
- **本研究の値**:
  - グラフェン: ε_s = -8.87, ε_p = 0.0, V_ssσ = -5.0, V_spσ = 4.7, V_ppσ = 5.5, V_ppπ = -2.7 (eV)
  - ダイヤモンド: ε_s = -2.99, ε_p = 3.71, V_ssσ = -1.938, V_spσ = 2.476, V_ppσ = 3.064, V_ppπ = -0.952 (eV)

### クボー公式 (Kubo formula)

- **概念**: 線形応答理論による光学伝導度の計算。
- **入力**: 固有値 E_nk, 速度行列要素 v_nm = ⟨n|∂H/∂k|m⟩
- **出力**: σ(ω)（周波数依存光学伝導度）

## 重要な関係性の整理

```
E(k) バンド構造
  ├── E_g（バンドギャップ）      ─── 分類軸1
  ├── ⟨v_α v_β⟩（速度テンソル） ─── D_eff ─── 分類軸2
  ├── σ(ω) → ε(ω) → R(ω)       ─── 完全な光学記述（固体）
  └── IPR（逆参加率）            ─── δ プロキシ（概念的）

固体系: E_g + D_eff ≡ ε(ω) の粗視化（同等の情報）
液体系: ε(ω) が定義困難 → E_g + D_eff（+ δ_nuc）が独自の手段
```
