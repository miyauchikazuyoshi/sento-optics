# 03: 相の連続的統一記述

## 1. 液体の再定義

**液体とは、構成粒子（原子核）の位置が非局在化し、かつ粒子間相互作用により有限体積に拘束された状態である。**

### 従来定義との比較
- 従来：「せん断応力に対して連続変形する」（何をするかの記述）
- 本定義：「核が非局在化しているが拘束されている」（何であるかの記述）

### 本定義からの導出
- 表面平滑性 → 核の自由度 → エネルギー最小化の実行 → 平滑
- 光沢 → 核非局在化 → 電子環境均一化 → 光子の反射
- 流動性 → 核が格子点に拘束されない → 連続変形可能

従来定義からは光沢を導出できない。本定義からは導出可能。

## 2. 非局在化度δによる相の連続記述

δ_nucを横軸に取ると：

```
固体 → ガラス → 液体 → 湯気(霧) → 気体
(局在)  (閾値近傍) (非局在化) (閾値近傍) (拘束喪失)
```

- 固体→ガラス：δ_nucがLindemann閾値をギリギリ超える。長距離秩序喪失だが流動に至らない。
- ガラス→液体：δ_nucが十分に閾値を超え、流動が実現。
- 液体→湯気：拘束喪失の閾値付近。微小液滴が気相中に分散。局所的にδが閾値を超えたり超えなかったり。
- 湯気→気体：完全に拘束喪失。

### ガラスの位置づけ
ガラス転移が「相転移か否か」の論争は、離散的相分類を前提としている。
連続パラメータδで見れば、閾値近傍の緩和が極端に遅い領域として自然に記述。

## 3. 核と電子の二軸相図

| | δ_elec 低 | δ_elec 高 |
|---|---|---|
| **δ_nuc 低** | 絶縁体固体（ダイヤモンド） | 金属固体（Cu, Fe） |
| **δ_nuc 高** | 絶縁体液体（水） | 液体金属（Hg） |
| **δ_nuc 極大** | — | 超流動（He-4） |
| **δ_nuc 低, δ_elec 極大** | — | 超伝導 |

核が局在すると電子の非局在化が促進される（周期ポテンシャル→Bloch波）相補性。

## 4. 経路積分による統一

量子揺らぎと熱揺らぎは経路積分形式で同一枠組み。
δの起源が量子的か熱的かは連続パラメータの両極であり本質的区別ではない。

これにより：
- 通常液体（熱的δ_nuc）と超流動（量子的δ_nuc）が同軸上に乗る
- 金属（熱的δ_elec寄与あり）と超伝導（量子的δ_elec）が同軸上に乗る
