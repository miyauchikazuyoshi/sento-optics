# 表面張力の現象論的再記述 — δ_elec による Miedema の n_ws の起源解明

**Status: Paper 2 の理論的骨格を整理したメモ。**

---

## 1. 核心命題

### A. 上位命題
液体金属の表面張力 γ は、電子の非局在化度 δ_elec に支配される。

### B. 中位命題
Miedema の境界電子密度 n_ws は δ_elec のプロキシ（代理変数）である。
非局在化した電子ほど Wigner-Seitz セル境界まで均一に広がり、
n_ws が高くなる。

### C. 下位命題
δ_elec は逆参加率 (IPR) を通じて操作的に定義可能であり、
基底関数の選択に対してロバストなランキングを与える。

## 2. 因果チェーン

```
電子の軌道対称性 (s, p, d)
    ↓
波動関数の空間的広がり (IPR → δ_IPR)
    ↓
Wigner-Seitz セル境界での電子密度 (n_ws)
    ↓
表面エネルギー (Miedema: γ ∝ n_ws^{5/3} / V_m^{2/3})
    ↓
表面張力 γ
```

**Paper 2 が新たに提案するのは最初の矢印**（δ_IPR → n_ws）。
残りは既存の Miedema 関係。

## 3. なぜ既存理論では不十分か

### 3.1 ギャップの所在

| 理論 | 何ができるか | 何ができないか |
|------|-------------|---------------|
| Lang-Kohn ジェリウム | アルカリ金属の表面エネルギー | 高密度金属で負の表面エネルギー（破綻） |
| 安定化ジェリウム | 全金属で正の表面エネルギー | r_s のみの一変数理論 → Al と Zn を区別不能 |
| DFT スラブ計算 | ±10% の精度で γ を計算 | 個別の数値。なぜその値か説明する記述子がない |
| Miedema モデル | n_ws^{5/3}/V_m^{2/3} で γ を予測（±10-20%） | n_ws がなぜ金属ごとに異なるか未説明 |

### 3.2 Al/Zn 問題（中核的テストケース）

- Al: r_s = 2.07, γ = 1140 mN/m
- Zn: r_s = 2.12, γ = 782 mN/m
- Δr_s = 2.4%（ほぼ同じバルク密度）
- Δγ = 46%（まるで違う表面張力）

**ジェリウムでは区別不能。DFT では計算はできるが説明はできない。
Miedema では n_ws(Al) > n_ws(Zn) で説明できるが、なぜ n_ws が
違うかは未解明。**

δ の回答：Al は sp 金属（電子が非局在化 → セル境界まで広がる →
n_ws 高 → γ 高）。Zn は d 金属的（3d 電子が原子上に局在 →
セル境界に到達しにくい → n_ws 低 → γ 低）。

### 3.3 Williams et al. (1980) の 45年間の未解決問題

Williams, Gelatt, Moruzzi は PRL 44, 429 (1980) で
"Miedema の n_ws の物理的描像は不適切 (inappropriate)" と指摘。
n_ws が表面張力を予測できる理由の物理的説明を求めた。

**45 年間、この問いに直接答えた論文はない。**

δ_IPR は "n_ws は電子の非局在化度のプロキシである" という回答を与え、
この 45年来の問いに電子構造側から初めて答える。

## 4. sp 金属 vs d 金属の二分法

### 4.1 物理的メカニズム

**sp 金属**（Na, K, Al）：
- 価電子は s, p 軌道 → 方向性が弱い → 空間的に広がる
- δ_IPR が高い
- セル境界での電子密度 n_ws が高い
- 表面を作るとバルクからの電子のはみ出しが大きい → γ 大

**d 金属**（Cu, Zn, Fe）：
- 価電子の多くが d 軌道 → 方向性が強い → 原子上に局在
- δ_IPR が低い
- セル境界での電子密度 n_ws が低い
- 表面を作ってもバルクからのはみ出しが小さい → γ が sp 金属より
  r_s の割に低い

### 4.2 計算による確認

**ダイマー計算（11 元素 × 3 基底関数）：**
- δ_IPR と n_ws の相関: r = 0.84 (def2-QZVP), r = 0.72 (def2-SVP)
- ランキングはすべての基底関数で安定

**スラブ計算（5 金属、QE 7.5）：**
- n_mid/n_bulk 比（セル間領域の電子密度 / バルク平均密度）:
  - sp 金属: Na 1.09, K 1.13, Al 0.96（バルクとほぼ同じ = 均一）
  - d 金属: Cu 0.33, Zn 0.28（バルクの 1/3 = 強い局在）
  - sp/d 比: 3.5倍

**Wannier 関数（MLWF）：**
- 平均スプレッド: Al 12.9, Na 11.7 Å² >> Cu 1.9, Zn 0.7 Å²
- sp/d 比: 6-19倍

### 4.3 ablation study（アブレーション解析）

**全電子密度では sp > d の傾向が見えない。**
Na の全電子密度プロファイルでは、1s, 2s, 2p の芯電子が
原子位置にピークを作り、n_mid/n_bulk が低く見える。

**価電子のみに分解すると、sp > d の傾向が明瞭に現れる。**

→ 表面張力を決めているのは全電子密度ではなく**価電子の非局在化度**。
これは Miedema の n_ws が「境界での密度」であることと整合する：
芯電子はセル境界に到達しないため、n_ws は事実上の価電子密度。

## 5. n_ws の物理的起源（本研究の主張）

> **n_ws は、価電子の非局在化度 δ_elec の実空間的な現れである。**

- δ_elec が高い（sp 金属）→ 電子が均一に広がる → セル境界でも密度が高い → n_ws 大
- δ_elec が低い（d 金属）→ 電子が原子上に局在 → セル境界で密度が低い → n_ws 小

Miedema の γ ∝ n_ws^{5/3} / V_m^{2/3} が成功する理由：
n_ws は（意図せず）δ_elec を測定していた。

## 6. Paper 1 との統一

| Paper | 系 | δ_elec が制御するもの | メカニズム |
|-------|---|---------------------|-----------|
| Paper 1 | 固体炭素同素体 | 光学応答（光沢） | δ 高 → 集団的位相コヒーレント応答 → 鏡面反射 |
| Paper 2 | 液体金属 | 表面張力 | δ 高 → 電子がセル境界にはみ出す → 界面エネルギーコスト大 |

**同じ変数 δ_elec が、光学と表面張力という異なる界面現象を支配する。**

これは偶然ではない。どちらも**界面における電子密度の空間構造**の
帰結：
- 光学：界面で電子が集団応答できるか → 反射
- 表面張力：界面で電子がバルクから失われるか → エネルギーコスト

δ_elec が高い金属（Al）は、光沢も強く、表面張力も高い。
δ_elec が低い金属（Zn）は、光沢が弱く、表面張力も低い（r_s の割に）。

## 7. 既存理論との関係

本理論は既存理論を否定しない。補完する。

| 既存理論 | 本理論との関係 |
|---------|-------------|
| Lang-Kohn ジェリウム | r_s のみの理論。δ は r_s が同じでも異なる情報を提供 |
| 安定化ジェリウム | 同上 |
| DFT スラブ計算 | DFT は正確な γ を計算する。δ はなぜその値になるかを説明する |
| Miedema モデル | n_ws は δ のプロキシ。δ が n_ws の物理的起源を与える |
| Halas et al. (2002) | 「非局在化電子の整数個数」で s ブロック表面エネルギー。δ はこれを連続変数に一般化し、d 金属にも適用 |

## 8. 限界と将来課題

1. **δ_IPR と γ の直接相関は弱い** (r = 0.08)。δ → n_ws → γ の
   間接経路であり、直接予測子ではない。
2. **ダイマー計算の δ_IPR vs n_ws 相関** (r = 0.53) は moderate。
   δ は「説明的記述子」であり「予測的記述子」ではない。
3. **基底関数依存性**：δ_IPR の絶対値は基底に依存する（N_AO が変わる）。
   ランキングは安定 (r = 0.72-0.84) だが、普遍的な δ の値を定義する
   には基底に依存しない定式化（Wannier スプレッド、ELF 等）が必要。
4. **液体の δ_nuc への拡張**：固体スラブ計算の結果を液体に転用している。
   液体状態での AIMD 計算で δ_elec が変わるか検証が必要。
5. **元素の拡張**：現在 11 元素（ダイマー）+ 5 元素（スラブ）。
   遷移金属全体、ランタノイド、合金系への拡張が必要。

## 9. δ_IPR の操作的定義（Paper 2 版）

```
δ_IPR = 1 / (N_AO · ⟨IPR⟩_occ)

IPR_n = Σ_i |c_{ni}|⁴ / (Σ_i |c_{ni}|²)²
      = Σ_i (Mulliken weight of orbital i in state n)²

⟨IPR⟩_occ = (1/N_occ) Σ_{n ∈ occ} IPR_n
```

- N_AO: 原子中心基底関数の数
- c_{ni}: KS 軌道 n の基底関数 i への展開係数
- 占有状態のみで平均
- 完全非局在化: δ_IPR = 1, 完全局在化: δ_IPR = 1/N_AO → 0
